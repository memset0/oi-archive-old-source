# 问题描述

<div class="pdcont">　　火车从始发站（称为第1站）开出，在始发站上车的人数为a，然后到达第2站，在第2站有人上、下车，但上、下车的人数相同，因此在第2站开出时（即在到达第3站之前）车上的人数保持为a人。从第3站起（包括第3站）上、下车的人数有一定规律：上车的人数都是前两站上车人数之和，而下车人数等于上一站上车人数，一直到终点站的前一站（第n-1站），都满足此规律。现给出的条件是：共有N个车站，始发站上车的人数为a，最后一站下车的人数是m（全部下车）。试问x站开出时车上的人数是多少？</div>
# 输入格式

<div class="pdcont">　　输入包含一行， 有四个正整数：a（a&lt;=100），n（n&lt;=20），m（m&lt;=10000）和x（x&lt;=19）</div>
# 输出格式

<div class="pdcont">　　输出为一个整数，为x站开出时车上的人数。</div>
