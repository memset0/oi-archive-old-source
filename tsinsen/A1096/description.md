# 题面



<div class="pdcont"><b>【问题描述】</b><br/>
　　<b>       </b>求一个长度为n的字符串中所有长度为k的子字符串，并将所有子串按字典序从小到大输出。<br/>
<br/>
<b>【输入格式】</b><b></b><br/>
　　输入共2行。<br/>
　　第一行为字符串，其长度为n（1≤n≤1000），字符串中不包含空格。<br/>
　　第二行一个整数k，0&lt;k≤min(5, n)。<br/>
<b> </b><br/>
<b>【输出格式】</b><b></b><br/>
　　输出共一行。<br/>
　　按从小到大排列的所有子字符串, 子串之间用空格隔开。<br/>
<br/>
<b>【样例输入】</b><b></b><br/>
　　HelloWorld<br/>
　　3<br/>
<br/>
<b>【样例输出】</b><b></b><br/>
　　Hel Wor ell llo loW oWo orl rld</div>
