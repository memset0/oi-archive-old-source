# 问题描述

<div class="pdcont">　　香蕉3元/斤，橘子2元/斤，葡萄8元/斤。买a斤香蕉、b斤橘子、c斤葡萄，问需要多少钱？</div>
# 输入格式

<div class="pdcont">　　输入包括三行，每行一个0至200的整数，分别表示a,b,c。</div>
# 输出格式

<div class="pdcont">　　输出所需要的钱数。</div>
# 样例输入

<div class="pddata">1<br/>
2<br/>
3</div>
# 样例输出

<div class="pddata">31</div>
