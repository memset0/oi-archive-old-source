# 问题描述

<div class="pdcont">　　如果a是b的倍数，则b是a的约数。给定一个正整数n，求n的正约数的个数。</div>
# 输入格式

<div class="pdcont">　　输入包含一个正整数，为n。(1&lt;=n&lt;=10000)</div>
# 输出格式

<div class="pdcont">　　输出n的约数个数。</div>
# 样例输入

<div class="pddata">12</div>
# 样例输出

<div class="pddata">6</div>
