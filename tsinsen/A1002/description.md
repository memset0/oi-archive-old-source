　　000001<br/>
　　000010<br/>
　　000011<br/>
　　000100<br/>

　　000001<br/>
　　000010<br/>
　　000100<br/>
　　000111<br/>
　　001000<br/>
　　001011<br/>
　　请按从小到大的顺序输出这些01串。</div>
# 输出格式

<div class="pdcont">　　每行一个01串。</div>
