题目描述
<small>Description</small>
</span></div>
<div class="panel-body">

<p>Fibonacci词定义如下：FIB1 = b  FIB2 = a 当k &gt;= 1时, FIBk+2 = FIBk+1*FIBk <br>其中* 表示词的连接。因此，我们有：</p>
<p> FIB3 = ab;     FIB4 = aba;    FIB5 = abaab;     FIB6 = abaababa.</p>
<p>问题是，对给定的词，问该词在FIBn中出现多少次。 </p>

</div>
</div>

<div class="panel panel-default">
<div class="area-title">
<span>
输入描述
<small>Input Description</small>
</span></div>
<div class="panel-body">
<p>输入有两行，第一行为给出的只含有a 或 b的词，长度不超过30.第二行给出一个正整数n， n &lt;= 200.</p>

</div>
</div>
<div  class="panel panel-default">
<div class="area-title">
<span>
输出描述
<small>Output Description</small>
</span></div>
<div class="panel-body">

<p class="p0">对给定的词，输出在FIBn中出现的次数&nbsp;</p>

</div>
</div>


<div class="panel panel-default">
<div class="area-title">
<span>
样例输入
<small>Sample Input</small>
</span></div>
<div class="panel-body">
<p>aba<br>6 </p>

</div>
</div>

<div class="panel panel-default">
<div class="area-title">
<span>
样例输出
<small>Sample Output</small>
</span></div>
<div class="panel-body">
<p>3</p>

</div>
</div>
