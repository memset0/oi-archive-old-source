题目描述
<small>Description</small>
</span></div>
<div class="panel-body">

<p><span style="">FJ的N头奶牛有一些共同之处(1 &lt;= N &lt;= 100,000)。FJ可以将这N头奶牛通过K种特征来归类(1 &lt;= K &lt;= 30)。例如，一些奶牛表现出来的特征1可能是有斑点，特征2可能是较之于PASCAL更喜欢C，等等。</span></p>
<p><span style=""><span style="">FJ发明了一种简明的描述特征的方法——“特征码”，用一个长度为k的二进制串来表示这头牛的特征表现。例如，一头牛的“特征码”为13，转换为二进制就是1101，代表这头牛具有特征1、3、4 (从右读到左)，但是不表现特征2。总的说来，如果这头奶牛表现特征i，那么我们在他的“特征码”的二进制的第i位就为1。</span></span></p>
<p><span style=""><span style=""><span style="">FJ将奶牛排成了一个1..N的队列，他注意到一种确定的排列方法可以使奶牛们的表现更“平衡”。一个连续的i..j的范围平衡表示为如果K种特征都有同样多的奶牛来表现。FJ想知道他究竟可以排出一个多长的“平衡”队列。请帮助他。</span></span></span></p>

</div>
</div>
