题目描述
<small>Description</small>
</span></div>
<div class="panel-body">

<p>在一个<span style="font-family: 'Times New Roman';">01</span><span style="">矩阵中，包含有很多的正方形子矩阵，现在要求出这个</span><span style="font-family: 'Times New Roman';">01</span><span style="">矩阵中，最大的正方形子矩阵，使得这个正方形子矩阵中的某一条对角线上的值全是</span><span style="font-family: 'Times New Roman';">1</span><span style="">，其余的全是</span><span style="font-family: 'Times New Roman';">0</span><span style="">。</span></p>

</div>
</div>

<div class="panel panel-default">
<div class="area-title">
<span>
输入描述
<small>Input Description</small>
</span></div>
<div class="panel-body">
<p>第一行有两个整数n和m（<span style="font-family: 'Times New Roman';">1&lt;=</span>n,m&lt;=1000<span style="">）。接下来的</span>n行，每行有m个<span style="font-family: 'Times New Roman';">0</span><span style="">或</span><span style="font-family: 'Times New Roman';">1</span><span style="">的数字。每两个数字之间用空格隔开。</span></p>
<p> </p>

</div>
</div>
<div  class="panel panel-default">
<div class="area-title">
<span>
输出描述
<small>Output Description</small>
</span></div>
<div class="panel-body">

<p class="p0">只有一个整数，即这个满足条件的最大的正方形子矩阵的边长。</p>

</div>
</div>


<div class="panel panel-default">
<div class="area-title">
<span>
样例输入
<small>Sample Input</small>
</span></div>
<div class="panel-body">
<p>4 6</p>
<p>0 1 0 1 0 0</p>
<p>0 0 1 0 1 0</p>
<p>1 1 0 0 0 1</p>
<p>0 1 1 0 1 0</p>

</div>
</div>

<div class="panel panel-default">
<div class="area-title">
<span>

<small>Sample Output</small>

<p>3</p>
