
# 题目描述

4qwerty7 回想起一道题目 <http://uoj.ac/problem/4>。

请实现它的 checker。

# 输入格式

原题输入后紧随一个回车，然后是原题输出内容。

# 输出格式

数据保证输入合法，请输出最终得分。

# 样例

#### 输入样例

```plain
3 3 100 2 3 1 1 0 
2 1 1 
2 3 3 
4 7 1
4
2 2 2 3 2
2 3 1 3 2
2 2 1 3 1
3 1 3 2 3 3 3
```

#### 输出样例

```plain
15
```
