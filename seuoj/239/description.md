
# 题目描述

输入正整数 $n,l,r$ 找到长度为 $n$ 的所有公比不为1的且每个元素都在 $[l,r]$ 内的等比数列。

# 输入格式

第一行一个整数 $T(1\leq T\leq 5)$ 代表数据组数。

每组数据一行三个正整数 $n,l,r(2\leq n\leq 10^7,1 \leq l \leq r 10^7)$。

# 输出格式

每组数据一行代表不同的等比数列个数

# 样例

#### 输入样例

```plain
1
3 3 10
```

#### 输出样例

```plain
2
```
#### 输出解释

满足条件的等比数列有 $4 6 9$ 和 $9 6 4$。
