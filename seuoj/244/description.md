
# 题目描述

> 这个世界本没有内鬼，或者人人都是内鬼。

> <div style="text-align:right">雅米·小</div>

现在有 $n$ 个人在一起，每个人都有一个编号，其中有一个内鬼。内鬼不会分身，其他人都会分身 $1$ 次，现在给出分身后的结果，请你帮小雅米抓出内鬼。

# 输入格式

第一行输入一个整数 $n(2 \leq n \leq 10^6)$，表示 $n$ 个人。

第二行输入 $2n - 1$ 个整数，表示每个人的编号 $x(1 \leq x \leq 10^9)$。

# 输出格式

输出一个数 $x$，表示内鬼的编号。

# 样例

#### 样例输入

```plain
3
1 1 2 2 3
```

#### 样例输出

```plain
3
```
