# 题目描述

<p>这里是NOI笔试模拟题，采用提交答案的方式提交。</p>
<p><strong>比赛时如果未得 $0$ 分则显示的分数即为满分。</strong></p>
<p>题目可以下载，也可以直接阅读下面的题面。</p>
<p>作答方式是提交一个答案文件，包含$51$行，每行包含选择的序号。</p>
<p>$1 \sim 40$是单选题，$41 \sim 51$是多选题，每道题的答案是一个由大写字母组成的无重复字符串，字母按照升序排列，表示你的选项。</p>
<p>评分方式，初始时你的得分为$100$分，将比较$1 \sim 50$题你的答案和标准答案，若不一样则扣一分。<strong>注意：第$51$题不计入最后得分</strong>。</p>
<p><strong>请注意对每一行，即使不会做这道题，也请输出一个非空的由大写字母组成的字符串，否则会造成评分时题号错位。</strong></p>
<p>例如如下题目：</p>

<pre><code>A
BDE</code></pre>
