# 题目描述

<p>给你两个数 $a$ 和 $b$，请输出他们的和。</p>

# 输入格式


<p>一行，两个用空格隔开的整数 $a$ 和 $b$。</p>

# 输出格式


<p>一个整数，表示 $a + b$。</p>

# 样例一


<h4>input</h4>
<pre>2 3

</pre>

<h4>output</h4>
<pre>5

</pre>


# 限制与约定


<p>$0 \leq a, b \leq 10^9$</p>
<p><strong>时间限制</strong>：$1\texttt{s}$</p>
<p><strong>空间限制</strong>：$256\texttt{MB}$</p>
