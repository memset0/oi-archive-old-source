# 题目描述

<p>你正在玩一个关于长度为 $n$ 的非负整数序列的游戏。这个游戏中你需要把序列分成 $k + 1$ 个非空的块。为了得到 $k + 1$ 块，你需要重复下面的操作 $k$ 次：</p>
<ol><li>选择一个有超过一个元素的块（初始时你只有一块，即整个序列）</li>
<li>选择两个相邻元素把这个块从中间分开，得到两个非空的块。</li>
</ol><p>每次操作后你将获得那两个新产生的块的元素和的乘积的分数。你想要最大化最后的总得分。</p>

# 输入格式


<p>第一行包含两个整数 $n$ 和 $k$。保证 $k + 1 \leq n$。</p>
<p>第二行包含 $n$ 个非负整数 $a_1, a_2, \dots, a_n$ $(0 \leq a_i \leq 10^4)$，表示前文所述的序列。</p>

# 输出格式


<p>第一行输出你能获得的最大总得分。</p>
<p>第二行输出 $k$ 个介于 $1$ 到 $n - 1$ 之间的整数，表示为了使得总得分最大，你每次操作中分开两个块的位置。第 $i$ 个整数 $s_i$ 表示第 $i$ 次操作将在 $s_i$ 和 $s_{i + 1}$ 之间把块分开。</p>
<p>如果有多种方案使得总得分最大，输出任意一种方案即可。</p>

# 样例一


<h4>input</h4>
<pre>7 3
4 1 3 4 0 2 3

</pre>

<h4>output</h4>
<pre>108
1 3 5

</pre>

<h4>explanation</h4>
<p>你可以通过下面这些操作获得 $108$ 分：</p>
<ol><li>初始时你有一块 $(4, 1, 3, 4, 0, 2, 3)$。在第 $1$ 个元素后面分开，获得 $4 \times (1 + 3 + 4 + 0 + 2 + 3) = 52$ 分。</li>
<li>你现在有两块 $(4), (1, 3, 4, 0, 2, 3)$。在第 $3$ 个元素后面分开，获得 $(1 + 3) \times (4 + 0 + 2 + 3) = 36$ 分。</li>
<li>你现在有三块 $(4), (1, 3), (4, 0, 2, 3)$。在第 $5$ 个元素后面分开，获得 $(4 + 0) \times (2 + 3) = 20$ 分。</li>
</ol><p>所以，经过这些操作后你可以获得四块 $(4), (1, 3), (4, 0), (2, 3)$ 并获得 $52 + 36 + 20 = 108$ 分。</p>

# 限制与约定


<p>第一个子任务共 11 分，满足 $1 \leq k &lt; n \leq 10$。</p>
<p>第二个子任务共 11 分，满足 $1 \leq k &lt; n \leq 50$。</p>
<p>第三个子任务共 11 分，满足 $1 \leq k &lt; n \leq 200$。</p>
<p>第四个子任务共 17 分，满足 $2 \leq n \leq 1000, 1 \leq k \leq \min\{n - 1, 200\}$。</p>
<p>第五个子任务共 21 分，满足 $2 \leq n \leq 10000, 1 \leq k \leq \min\{n - 1, 200\}$。</p>
<p>第六个子任务共 29 分，满足 $2 \leq n \leq 100000, 1 \leq k \leq \min\{n - 1, 200\}$。</p>
<p><strong>时间限制：</strong>$2\texttt{s}$</p>
<p><strong>空间限制：</strong>$256\texttt{MB}$</p>
