 # 输入格式 
<p>
第一行有两个整数n和m（1≤n,m≤32），描述池塘规模。接下来的n行，每行有m个数字（非“0”即“1”）。每两个数字之间用空格隔开。</p> 

 
 # 输出格式 
<p>
对于每组输入数据，输出一行：一个数字，所需挡板的最短总长度。</p>
