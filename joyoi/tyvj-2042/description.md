 # 题目描述 
有N条线段，已知每条线段的起点和终点（50000以内），然后有M个询问，每次询问一个点（50000以内），求这个点在多少条线段上出现过？<br> 

 
 # 输入格式 
第一行N线段条数<br>接下来N行，每行两个数，线段的起点和终点<br>第N+2行一个数M询问个数<br>接下来M行，每行一个点<br><br> 

 
 # 输出格式 
对于每个询问，求答案<br> 

 
 # 提示 
N,M&lt;=50000<br>南雅中学<br>
