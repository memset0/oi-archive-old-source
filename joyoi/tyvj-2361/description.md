 # 题目描述 
<p>
给你一棵TREE,以及这棵树上边的距离.问有多少对点它们两者间的距离小于等于K<br></p> 

 
 # 输入格式 
<p>
N（n<=40000）<br>接下来n-1行边描述管道，按照题目中写的输入<br>接下来是k<br></p> 

 
 # 输出格式 
<p>
一行，有多少对点之间的距离小于等于k<br></p>
