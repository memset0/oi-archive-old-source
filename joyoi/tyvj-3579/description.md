 # 题目描述 
<p>
给出一个具有N个结点的无向图,将其分成两个集合S1,S2.<br>这两个集合的点的个数一样多,但连接它们的边最少.</p> 

 
 # 输入格式 
<p>
第一行给出数字N,M,代表有N个点,M条边.<br>下面M行,每行两个数字代表此两点间有条边.</p> 

 
 # 输出格式 
<p>
按升序输出S1中的那N/2个点的编号.</p> 

 
 # 提示 
<p>
N<=26</p>
