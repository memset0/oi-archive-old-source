 # 题目描述 
给定一个长度为n的数列{a1,a2...an}，每次可以选择一个区间[l,r]，使这个区间内的数都加一或者都减一。<BR>问至少需要多少次操作才能使数列中的所有数都一样,并求出在保证最少次数的前提下,最终得到的数列有多少种。 

 
 # 输入格式 
第一行一个正整数n&nbsp;<BR>接下来n行,每行一个整数,第i+1行的整数表示ai。 

 
 # 输出格式 
第一行输出最少操作次数<BR>第二行输出最终能得到多少种结果 

 
 # 提示 
对于100%的数据，n=100000，0&lt;=ai&lt;2147483648。
