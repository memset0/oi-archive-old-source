 # 输入格式 
<p>
第一行两个正整数N、S，分别表示小Y 能预知的天数以及初始时拥有的钱数。<br>接下来N 行，第K 行三个实数AK、BK、RateK，意义如题目中所述</p> 

 
 # 输出格式 
<p>
只有一个实数MaxProfit，表示第N 天的操作结束时能够获得的最大的金钱<br>数目。答案保留3 位小数。</p>
