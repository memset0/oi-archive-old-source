 # 题目描述 
<p>
<br>贝西喜欢棋盘游戏和角色扮演类游戏所以她说服Farmer John把她带到玩具店，在那里，她购买了三个不同的骰子，这三个质量均匀的骰子，分别有S1,S2,S3个面。(2 <= S1 <= 20; 2 <= S2 <= 20; 2 <= S3 <= 40).<br><br>贝西掷啊掷啊掷啊，想要知道出现几率最大的和是多少。<br><br>问题给出三个骰子的面数，让你求出出现几率最大的和是多少。如果有很多种和出现的几率相同，那么就输出小的那一个。<br><br></p> 

 
 # 输入格式 
<p>
*第一行：三个由空格隔开的整数：s1,s2,s3<br><br></p> 

 
 # 输出格式 
<p>
<br>*第一行：所要求的解<br><br></p>
