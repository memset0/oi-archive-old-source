 # 题目描述 
<p>
给你N对数A1,B1……An,Bn 。要求你从中找出最多的对，把它们按照一种方式排列，重新标号1,2,..,k。<br>能满足对于每一对 i < j,都有 Ai > Bj。<br></p> 

 
 # 输入格式 
<p>
第一行给出一个数字N,N < = 100000<br>下面N行，分别给出Ai,Bi,其小于10^9</p> 

 
 # 输出格式 
<p>
输出K的极大值</p>
