 # 题目描述 
<p>
Problem 1 : binary<br>二进制除法<br><br>问题描述：<br>　　二进制数n mod m的结果是多少？<br><br></p> 

 
 # 输入格式 
<p>
　　第一行输入一个二进制数n。<br>　　第二行输入一个二进制数m。<br><br></p> 

 
 # 输出格式 
<p>
　　输出n mod m的结果。<br><br></p>
