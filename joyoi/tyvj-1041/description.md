 # 题目描述 
给出一个表达式,其中运算符仅包含+,-,要求求出表达式的最终值<BR><BR>保证数据中不会出现负数，并且同时保证，如果你按从左到右的顺序计算，同样也不会出现负数的情况。<BR><BR> 

 
 # 输入格式 
仅一行，即为表达式<BR> 

 
 # 输出格式 
仅一行，既为表达式算出的结果 

 
 # 提示 
表达式总长度&lt;=255<BR>表达式中数字位数&lt;=255
