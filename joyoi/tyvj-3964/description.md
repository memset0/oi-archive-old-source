 # 题目描述 
<p>输入n个学生的编号、语文、数学、英语的成绩，求总分第一名的同学的编号。</p> 

 
 # 输入格式 
<p>3</p>

<p>1&nbsp;90&nbsp;100&nbsp;100</p>

<p>2&nbsp;70&nbsp;80&nbsp;89</p>

<p>3&nbsp;97&nbsp;98&nbsp;100</p> 

 
 # 输出格式 
<p>3</p>

<p>&nbsp;</p> 

 
 # 提示 
<h1 style="font-style: italic;"><u><strong>如果有多个人并列第一，那么打印最先的那个人！</strong></u></h1>
