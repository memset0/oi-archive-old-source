 # 题目背景 
咱们的涂二同学开始学直线方程了，特此纪念……<BR> 

 
 # 题目描述 
输入一个直线的一般式方程与一个点的坐标（两行且严格按照样例形式）,请输出该直线关于该点对称的直线的一般式方程。<BR> 

 
 # 输入格式 
一个直线的一般式方程与一个点的坐标.<BR> 

 
 # 输出格式 
一行，即直线的一般式方程,请保证系数为最简整数。<BR> 

 
 # 提示 
保证该点不在直线上,系数均为整数(但不排除系数为零的可能),各系数与坐标均不超过10000。<BR>输出请保证x系数为正,x系数为0则y系数为正。
