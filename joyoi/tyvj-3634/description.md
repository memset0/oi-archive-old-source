 # 输入格式 
<p>
只包含4个整数，它们彼此用空格隔开，分别为xp,yp,xs,ys。并且它们的都小于10000000。</p> 

 
 # 输出格式 
<p>
含一个整数，表示从点p到点s至少需要经过的马步移动次数。</p>
