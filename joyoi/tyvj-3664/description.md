 # 题目描述 
<p>
<br>对于任意输入的正整数n，请编程求出具有n个不同因子的最小正整数m。<br>例如：n=4，则m=6，因为6有4个不同整数因子1，2，3，6；而且是最小的有4个因子的整数。<br></p> 

 
 # 输入格式 
<p>
n（1≤n≤50000）<br></p> 

 
 # 输出格式 
<p>
m<br></p>
