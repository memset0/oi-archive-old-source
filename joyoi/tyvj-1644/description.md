 # 题目背景 
柯桥中学60周年校庆欢乐赛第3题 

 
 # 题目描述 
&nbsp;&nbsp;&nbsp;&nbsp;问题概括：n个元素的排序一共有n!个，询问排列中逆序对个数为m个的排列有多少个。 

 
 # 输入格式 
输入文件仅一行有两个正整数n，m 

 
 # 输出格式 
仅有一行,即满足要求的排列的个数 

 
 # 提示 
对于20%的数据，n&lt;=10<BR>对于40%的数据，n&lt;=20<BR>对于100%的数据，1&lt;=n&lt;=50&nbsp;&nbsp;m&lt;=(n-1)*n/2
