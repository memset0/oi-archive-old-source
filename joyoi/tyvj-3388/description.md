 # 题目描述 
<p>
　　给出n和m（ m <= n ），生成数字1..n中取m个数的所有组合，按字典序由小到大输出。例如：输入n=3，m=2，则输出：<br>1 2<br>1 3<br>2 3<br></p> 

 
 # 输入格式 
<p>
　　一行，两个数，中间一个空格，分别为n和m。（ 1 <= m <= n <= 9 ）</p> 

 
 # 输出格式 
<p>
　　按字典序由小到大输出所有组合，一行一个组合样例。</p>
