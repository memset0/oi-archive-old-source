 # 题目描述 
<p>
给出一个数列C1,C2...Cn,求一个最小的Q,使得存在一个P(P<Q).<br>满足P / Q的小数形式连续包含这一段C1...Cn<br><br></p> 

 
 # 输入格式 
<p>
第一行一个数N,接下来N行,每行一个数,表示Ci<br></p> 

 
 # 输出格式 
<p>
输出一共2行,第一行表示P,第二行表示Q<br><br></p> 

 
 # 提示 
<p>
N<=100<br>P,Q不超过60位<br></p>
