 # 题目描述 
设a，b是给定的正整数。现有一机器人沿着一个有n级的楼梯上下。机器人每上升一次，恰好上升a级楼梯，每下降一次恰好下降b级楼梯。为使机器人经过若干次上下升降后，可以从地面到达楼梯顶，然后再返回地面，请给出n的最小值。<BR> 

 
 # 输入格式 
第一行包括两个正整数，由一个空格分开，1&lt;=a,b&lt;=1000000000。<BR> 

 
 # 输出格式 
仅有一行，包括你所得到的最小值。输入数据保证这个最小值不超过长整型。<BR>
