 # 题目描述 
<p>
　　输入N个整数,用快速排序法对它们进行从大到小排序, 输出排序后的结果。</p> 

 
 # 输入格式 
<p>
　　二行，第一行，n，表示有n（小于100000）个待排序数据。第二行，待排序数据的n个数据，每个数据间有一个空格隔开。</p> 

 
 # 输出格式 
<p>
　　一行，排好序的n个数据，每个数据间用一个空格间隔。</p>
