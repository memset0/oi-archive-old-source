 # 题目描述 
<p>
给出一个长度为n(n ≤ 1000)的正整数序列，求一个子序列，使得原序列中任意长度为m的子串中被选出的元素不超过k(k ≤ m ≤ 10)个，并且选出的元素之和最大。</p> 

 
 # 输入格式 
<p>
输入文件best.in的第一行三个数为n, m, k。第二行n个数，表示各元素数值大小。</p> 

 
 # 输出格式 
<p>
输出文件best.out只有一个数，表示最大元素和。</p>
