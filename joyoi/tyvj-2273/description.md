 # 题目描述 
<p>
寻找一个凸多边形，使它的边都具有给定的长度在这个任务里，我们说一个多边形是凸的，当且仅当他的每个内角严格的大于0度，且严格的小于180度<br></p> 

 
 # 输入格式 
<p>
第一行给出数字N.N小于等于1000<br>下面N行,每行一个数字代表边长,其小于等于10000</p> 

 
 # 输出格式 
<p>
输出N个点的坐标.</p> 

 
 # 提示 
<p>
请勿提交</p>
