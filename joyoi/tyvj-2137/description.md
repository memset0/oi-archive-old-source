 # 题目描述 
<p>输入n个数，判断是否为最简。最简输出YES，否则输出最简形式。</p>

<p>每个数的绝对值不超过10^255。</p> 

 
 # 输入格式 
<p>第一行，一个数n</p>

<p>接下来n个数(n&lt;=20)</p> 

 
 # 输出格式 
<p>N行，每行YES或最简形式</p> 

 
 # 提示 
<p>需要考虑的情况有：</p>

<p>1.多余的前导零；</p>

<p>2.多余的后导零;</p>

<p>3.多余的正负号；</p>

<p>另外，注意看清题目。</p>
