 # 题目描述 
<p>
小H有n个碗需要放进橱柜，她希望将他们叠起来放置。你知道每个碗都是规则的圆柱体，并且都是上宽下窄，你已经测量出了每个碗的两个半径及高，请你帮小H找出一种叠放顺序，使得叠放出来的碗堆的高度尽量小，比如：<br><br> <img border="0" src="/source/joyoi/tyvj-2661/img/aHR0cDovL3d3dy5qb3lvaS5jbi9wcm9ibGVtL3R5dmotMjY2MS9wcm9ibGVtc19pbWFnZXMvMzEyOS8xODU5LmpwZw==.jpg"> <br><br></p> 

 
 # 输入格式 
<p>
第一行一个整数n，表示碗的数目。以下n行，每行三个整数h，r1，r2。分别表示碗高及两个半径。其中r1<r2。<br><br></p> 

 
 # 输出格式 
<p>
仅一个数，表示最小的高度。答案四舍五入取整。<br><br></p> 

 
 # 提示 
<p>
数据范围：100%数据满足n < = 9。所有输入的数绝对值不超过1000。<br></p>
