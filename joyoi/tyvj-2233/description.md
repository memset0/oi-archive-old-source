 # 题目描述 
<p>
求N!的最后一位非零数<br></p> 

 
 # 输入格式 
<p>
本题有多组数据，每行一个数N (N<=10^100)<br></p> 

 
 # 输出格式 
<p>
对于每一个输入的N,输出N!的最后一位非零数<br><br></p>
