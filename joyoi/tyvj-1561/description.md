 # 题目描述 
将一个整数m分解为n个四次方数的和的形式，要求n最小。 

 
 # 输入格式 
一行，一个整数m。 

 
 # 输出格式 
一行，一个整数n。 

 
 # 提示 
对样例的解释：m=706,706=5^4+3^4,则n=2。<BR><BR>数据范围：<BR>对于30%的数据，m&lt;=5000;对于100%的数据，m&lt;=100,000大连24中翠神原创
