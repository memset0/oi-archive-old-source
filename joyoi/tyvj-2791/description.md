 # 题目描述 
<p>
Yallisha是一个很节俭的人——她非常喜欢买打折的东西。<br>经过长期的观察，她发现了各大超市会轮流进入打折阶段。因为周期并不一样，所以会出现某段时间没有超市打折，但某些时候所有超市一齐打折。<br>而现在，就是所有超市均打折出售优质商品的时候！<br>Yallisha当然不会放过这个机会了。<br>今天一早，她就决定去超市进行疯狂购物。<br>当然，因为Yallisha十分精明，在超市i购物一小时以后，下一小时可以得到的好商品价值会减少Bi，而每个超市第一次选购一小时所买到的好商品价值Ai也是不一样的。<br>因为当地政府的某些安排，有一条公交线路把各大超市都连了起来，因此Yallisha决定只乘坐这辆公交车来进行购物。Yallisha是十分精明的，因此等车时间可以忽略不计。在这个情况下，从超市x到超市y所需的时间是5*|x-y|分钟。假设Yallisha的家在第一间超市隔壁&#61514;。<br>为了尽可能在最短的时间内得到最多的获益，Yallisha希望计算出在m小时内能选购到的好商品价值最大是多少。<br>注意，如果剩下的时间不到1小时，那么是不能购物的。<br></p> 

 
 # 输入格式 
<p>
输入数据的第一行为两个整数n，m，表示有n个超市，要购物m小时。<br>第二行n个整数，第i个整数为Ai。<br>第三行n个整数，第i个整数为Bi。<br></p> 

 
 # 输出格式 
<p>
输出数据仅包含一行，为最大获益。</p> 

 
 # 提示 
<p>
数据范围：<br>对于30％的数据，1<=n<=1000，1<=m<=500<br>对于另外30％的数据，1<=n<=500，1<=m<=1000<br>对于另外40％的数据，1<=n<=1000，1<=m<=1000（传说中的疯狂购物……1000小时。。大家理解成每天8小时吧，呵呵……充要条件是Yallisha是个有钱有闲的家伙。。）<br></p>
