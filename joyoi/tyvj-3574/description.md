 # 题目描述 
<p>
Byteotia城市有n个 towns m条双向roads. <br>每条 road 连接 两个不同的 towns ,没有重复的road. <br>所有towns连通。<br></p> 

 
 # 输入格式 
<p>
输入n<=100000 m<=500000及m条边</p> 

 
 # 输出格式 
<p>
输出n个数，代表如果把第i个点去掉，将有多少对点不能互通。</p>
