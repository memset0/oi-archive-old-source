# 题目描述
这个数列的前五个数是$\begin{Bmatrix}0&0&1&1&2& \cdots \end{Bmatrix}$，你需要输出这个数列的第n个数。
# 输入格式
一行一个整数n。
# 输出格式
一行一个整数，代表这个数列的第n个数。
# 提示与说明
由于这个数列太特殊了，于是PM决定让n满足n$\leq$16
