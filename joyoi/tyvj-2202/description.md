 # 题目描述 
<p>
给出正整数n和k，计算j(n, k)=k mod 1 + k mod 2 + k mod 3 + … + k mod n的值，其中k mod i表示k除以i的余数。<br>例如j(5, 3)=3 mod 1 + 3 mod 2 + 3 mod 3 + 3 mod 4 + 3 mod 5=0+1+0+3+3=7<br></p> 

 
 # 输入格式 
<p>
输入仅一行，包含两个整数n, k。<br></p> 

 
 # 输出格式 
<p>
输出仅一行，即j(n, k)。<br><br></p> 

 
 # 提示 
<p>
50%的数据满足：1<=n, k<=1000<br>100%的数据满足：1<=n ,k<=10^9<br></p>
