 # 题目描述 
<p>
假设你有一条长度为5的木版，初始时没有涂过任何颜色。你希望把它的5个单位长度分别涂上红、绿、蓝、绿、红色，用一个长度为5的字符串表示这个目标：RGBGR。<br>每次你可以把一段连续的木版涂成一个给定的颜色，后涂的颜色覆盖先涂的颜色。例如第一次把木版涂成RRRRR，第二次涂成RGGGR，第三次涂成RGBGR，达到目标。<br>用尽量少的涂色次数达到目标。<br></p> 

 
 # 输入格式 
<p>
输入仅一行，包含一个长度为n的字符串，即涂色目标。字符串中的每个字符都是一个大写字母，不同的字母代表不同颜色，相同的字母代表相同颜色。<br><br></p> 

 
 # 输出格式 
<p>
仅一行，包含一个数，即最少的涂色次数。<br></p> 

 
 # 提示 
<p>
40%的数据满足：1<=n<=10<br>100%的数据满足：1<=n<=50<br></p>
