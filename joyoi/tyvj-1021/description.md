 # 题目描述 
数轴上有N个点，任意两点连线得到n(n-1)条线段，试求线段的总长。<BR> 

 
 # 输入格式 
第一行，一个整数N，表示点数。<BR>接下来N行，每行一个整数X_i，表示点的坐标。<BR> 

 
 # 输出格式 
一个整数，表示线段的总长。<BR> 

 
 # 提示 
N&nbsp;&lt;=&nbsp;10000&nbsp;,&nbsp;0&nbsp;&lt;=&nbsp;X_i&nbsp;&lt;=&nbsp;1000000000<BR>感谢！<BR>Vivian&nbsp;Snow<BR>
