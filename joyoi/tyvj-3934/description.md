 # 题目描述 
<p>去重排序<br />
&nbsp;任意输入n个整数，去掉重复的整数，然后把他们从小到大排列。<br />
&nbsp;</p> 

 
 # 输入格式 
<p>8<br />
14<br />
8<br />
21<br />
43<br />
8<br />
46<br />
43<br />
42<br />
&nbsp;</p> 

 
 # 输出格式 
<p>8<br />
14<br />
21<br />
42<br />
43<br />
46<br />
&nbsp;</p> 

 
 # 提示 
<p>【输入格式】<br />
&nbsp;第1行一个数n.<br />
&nbsp;第2行之第n+1行，每行一个数。<br />
【输出格式】<br />
从小到大排列的且不重复出现的整数<br />
&nbsp;</p>
