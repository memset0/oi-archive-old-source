 # 题目描述 
<p>
给出某两个整数a和b（a<=b）的最大公约数GCD和最小公倍数LCM，请找出满足的a和b，使得b-a的值最小。</p> 

 
 # 输入格式 
<p>
输入数据只有一行，包括两个整数GCD和LCM。输入保证至少存在一组解。</p> 

 
 # 输出格式 
<p>
输出包含一个整数，为最小的b-a的值。</p> 

 
 # 提示 
<p>
对于50%的数据，1<=a<=b<=10^3。<br>对于100%的数据，1<=a<=b<=10^9。<br></p>
