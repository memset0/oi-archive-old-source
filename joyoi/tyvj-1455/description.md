 # 题目描述 
&nbsp;&nbsp;&nbsp;&nbsp;贾老二是个品学兼优的好学生，但由于智商问题，算术学得不是很好，尤其是在解方程这个方面。虽然他解决&nbsp;2x=2&nbsp;这样的方程游刃有余，但是对于&nbsp;{x+y=3&nbsp;&nbsp;x-y=1}&nbsp;这样的方程组就束手无策了。于是他要你来帮忙。前提是一次方程组且保证在integer的范围内可以处理所有问题。<BR> 

 
 # 输入格式 
&nbsp;&nbsp;&nbsp;&nbsp;第一行一个数字N（1≤N≤100）表示要求的未知数的个数，同时也是所给的方程个数。<BR>&nbsp;&nbsp;&nbsp;&nbsp;第2到N+1行，每行N+1个数。前N个表示第1到N个未知数的系数。第N+1个数表示N个未知数乘以各自系数后的加和。（保证有唯一整数解）<BR> 

 
 # 输出格式 
&nbsp;&nbsp;&nbsp;&nbsp;一行N个数，表示第1到N个未知数的值。<BR> 

 
 # 提示 
ASYZOI贾老二专题<BR>
