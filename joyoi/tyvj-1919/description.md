 # 题目描述 
给定A、B、C三根足够长的细柱，在A柱上放有2n个中间有孔的圆盘，共有n个不同的尺寸，每个尺寸都有两个相同的圆盘，注意这两个圆盘是不加区分的（下图为n=3的情形）。现要将这些圆盘移到C柱上，在移动过程中可放在B柱上暂存。要求：<BR>（1）每次只能移动一个圆盘；<BR>（2）A、B、C三根细柱上的圆盘都要保持上小下大的顺序；<BR>任务：设An为2n个圆盘完成上述任务所需的最少移动次数，对于输入的n，输出An。<BR><img src="/source/joyoi/tyvj-1919/img/aHR0cDovL3d3dy5qb3lvaS5jbi9wcm9ibGVtL3R5dmotMTkxOS8mbmJzcDtodHRwOi8veXQudHl2ai5jbjo4MDgwL1Byb2JsZW1JbWcvUDEwNjkuanBn.jpg" border=0 align=middle> 

 
 # 输入格式 
一个正整数n，表示在A柱上放有2n个圆盘。 

 
 # 输出格式 
仅一行，包含一个正整数,&nbsp;为完成上述任务所需的最少移动次数An。<BR><BR>
