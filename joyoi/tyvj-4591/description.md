 # 题目描述 
<p>小C手里有n个数，他并不想和你说话并把这些数扔给了你，请输出n的值。</p> 

 
 # 输入格式 
<p>输入若干行，每行一个正整数，以EOF结束。</p> 

 
 # 输出格式 
<p>输出一个整数，表示输入的数的个数。</p> 

 
 # 提示 
<p>对于100%的数据，给出的正整数不超过2^31-1，答案不超过1,000,000。</p>
