# 题目描述
输入三个数，计算并输出它们的平均值以及乘积。
# 输入
输入一行，3个整数，整数之间有一个空格。
# 输出
输出2行。
第一行：3个数的平均值，保留2位小数。
第二行：3个数的乘积。
