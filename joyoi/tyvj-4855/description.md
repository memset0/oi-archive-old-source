# 

 
 # 题目描述 
<p>输入a,b,c三数，输出a*b*c的值。</p> 

 
 # 输入格式 
<p>输入数据共一行，为a,b,c三数。</p> 

 
 # 输出格式 
<p>输出a*b*c，行尾包含一个换行。</p> 

 
 # 提示 
<p><span style="line-height: 20.8px;">样例</span>输入：</p>

<p>6&nbsp;3&nbsp;8</p>

<p><span style="line-height: 20.8px;">样例</span>输出：</p>

<p>144</p>

<p>&nbsp;</p>

<p>数据范围：a,b,c均小于等于100000.</p>
