 # 题目描述 
棋盘上A点有一个过河卒，需要走到目标B点。卒行走的规则：可以向下、或者向右。同时在棋盘上C点有一个对方的马，该马所在的点和所有跳跃一步可达的点称为对方马的控制点。因此称之为“马拦过河卒”。<BR>棋盘用坐标表示，A点(0,&nbsp;0)、B点(n,&nbsp;m)(n,&nbsp;m为不超过20的整数)，同样马的位置坐标是需要给出的。现在要求你计算出卒从A点能够到达B点的路径的条数，假设马的位置是固定不动的，并不是卒走一步马走一步。<BR> 

 
 # 输入格式 
一行四个数据，分别表示B点坐标和马的坐标。 

 
 # 输出格式 
一个数据，表示所有的路径条数。
