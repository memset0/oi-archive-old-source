# 题目描述
二百万合众国大军抵达西线，乔治国王希望知道他们领先于同盟国多少兵力。
# 输入格式：
一行五个数，代表五个国家的兵力 顺序：德、英、美、法、奥（实力顺序，不代表兵力数量）

# 输出格式：
一行一个数，代表领先的兵力# 样例数据
