 # 题目描述 
王女士是一个数学爱好者，这些天她对B数很感兴趣。<BR>B数的定义：能被13整除且本身包含字符串“13”的数。<BR>例如：130和2613是B数，但是143和2639不是B数。<BR>你的任务是计算1到n之间有多少个数是B数。<BR> 

 
 # 输入格式 
&nbsp;&nbsp;&nbsp;输入数据只有一个数，为N(1≤N≤10^15)&nbsp; 

 
 # 输出格式 
输出数据包含一行，为1到n之间B数的个数。
