 # 题目描述 
<p>求菲波拉契数列第n项。</p>

<p>第0项a0=0，接着a1=1，a2=a1+a0，a3=a2+a1，&hellip;&hellip;</p>

<p>提示：a(n)=a(n-1)+a(n-2)</p>

<p>输入<br />
输入一个正整数n（n&lt;30）</p>

<p>输出<br />
输出an的值</p>

<p>样例输入<br />
3<br />
样例输出<br />
2</p> 

 
 # 输入格式 
<p>8</p> 

 
 # 输出格式 
<p>21</p> 

 
 # 提示 
<p>第一第二个斐波那契数是1</p>

<p>第三个开始&nbsp;&nbsp;是前两项之和</p>

<p>&nbsp;</p>

<p>比如&nbsp;1&nbsp;1&nbsp;2&nbsp;3&nbsp;5&nbsp;这样的规律</p>
