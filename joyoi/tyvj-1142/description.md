 # 题目描述 
1000的阶乘1*2*3*...*1000结果是一个很大的数，求这么大的数末尾有多少个连续的零。&nbsp;<BR><BR> 

 
 # 输入格式 
本题有多组测试数据&nbsp;<BR><BR>每组数据含有一个正整数N。(N不大于....算了,无限大,OK?)&nbsp;<BR><BR> 

 
 # 输出格式 
输出一个整数，表示N!的末尾有多少个连续的零。
