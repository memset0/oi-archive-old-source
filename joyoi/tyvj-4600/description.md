 # 题目描述 
<p>给定平面n个黑点，如果平面一个边平行于坐标轴的矩形3个角是黑色的那么就把那个矩形的第4个角改成黑色，最后平面上将会有多少个黑点&nbsp;</p> 

 
 # 输入格式 
<p>第一行一个数n表示点数</p>

<p>接下来n行每行2个数分别表示第i个黑点的横坐标和纵坐标</p> 

 
 # 输出格式 
<p>一个数表示平面上最后的黑点数</p> 

 
 # 提示 
<p>样例输入</p>

<p>5</p>

<p>1&nbsp;1</p>

<p>2&nbsp;1</p>

<p>1&nbsp;2</p>

<p>2&nbsp;3</p>

<p>3&nbsp;2</p>

<p>样例输出</p>

<p>9</p>

<p>&nbsp;</p>

<p>&nbsp;</p>

<p>数据范围</p>

<p>30%的数据满足&nbsp;n&lt;=100</p>

<p>100%的数据满足n&lt;=2*10^5&nbsp;&nbsp;坐标范围-10^5..10^5</p>
