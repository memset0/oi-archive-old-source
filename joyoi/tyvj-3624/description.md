 # 题目描述 
<p>
定义一个数的digit-product是它的各个位上的数字的乘积,定义一个数的self-product是它本身乘以它的digit-product。编程求self-product在a和b之间的数的个数。<br></p> 

 
 # 输入格式 
<p>
两个整数a,b(0 ≤ a ≤ b < 1018)。<br></p> 

 
 # 输出格式 
<p>
一个整数，self-product在a和b之间的数的个数。<br></p>
