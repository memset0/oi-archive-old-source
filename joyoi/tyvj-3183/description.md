 # 题目描述 
<p>
　　丑数<br>　　&#8226; 一个数的所有素数因子都在集合{2, 3, 5, 7}的数称为ugly number。<br>　　&#8226; 求第n小的丑数。</p> 

 
 # 输入格式 
<p>
　　输入一行，一个整数值n（n <= 10^5 ）。　</p> 

 
 # 输出格式 
<p>
　　输出共一行，一个整数，表示求第n小的丑数。</p> 

 
 # 提示 
<p>
　　数据范围： <br>　　　30%数据，n <= 10^2。 <br>　　　50%数据，n <= 10^3。 <br>　　　100%数据，n <= 4*10^3。<br>　　　最终答案保证小于2^63。<br><br></p>
