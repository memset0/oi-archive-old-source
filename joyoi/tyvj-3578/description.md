 # 题目描述 
<p>
给一个n*n的地图，每个格子有一个价格，找一个矩形区域，使其价格总和位于[k,2k]<br></p> 

 
 # 输入格式 
<p>
输入k n（n<2000)和一个n*n的地图<br></p> 

 
 # 输出格式 
<p>
输出矩形的左上和右下的列-行坐标<br>或NIE<br></p>
