 # 题目背景 
<p>开学第一天</p> 

 
 # 题目描述 
<p>星星的老师给星星出了一道题目：天上有N颗星星，把它分成若干组，问有几种方案？结果模M。</p> 

 
 # 输入格式 
<p>两个正整数N，M<br />
(1&lt;=n&lt;=5000)&nbsp;(1&nbsp;&lt;=&nbsp;M&nbsp;&lt;=&nbsp;1000000007)</p> 

 
 # 输出格式 
<p>一个数，为方案数ans</p> 

 
 # 提示 
<h1>测试样例</h1>

<h3>输入</h3>

<blockquote>
<p>5&nbsp;1000</p>
</blockquote>

<p>&nbsp;</p>

<h3>输出</h3>

<blockquote>
<p>7</p>
</blockquote>

<h3>样例解释</h3>

<blockquote>
<p>5=5</p>

<p>&nbsp;=4+1</p>

<p>&nbsp;=3+2</p>

<p>&nbsp;=3+1+1</p>

<p>&nbsp;=2+2+1</p>

<p>&nbsp;=2+1+1+1</p>

<p>&nbsp;=1+1+1+1+1</p>
</blockquote>
