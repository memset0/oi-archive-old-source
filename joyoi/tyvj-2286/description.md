 # 题目描述 
<p>
给出N个等腰三角形,每个三角由x,y,m三个参数来确定,这三个参数确定三角形的三个点分别为 (x;y), (x+m;y) and (x; y+m)</p> 

 
 # 输入格式 
<p>
第一行给出数字N,N<=2000<br>下面N行给出Xi,Yi,Mi<br>-10^7<=xi<=10^7<br> -10^7<= Xi,yi<= 10^7<br> 0 < mi< 1000</p> 

 
 # 输出格式 
<p>
输出三角形覆盖的总面积,答案保留一位小数</p>
