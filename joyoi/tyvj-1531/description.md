 # 题目描述 
一本书的页码是从1-n编号的连续整数：1,&nbsp;2,&nbsp;3,&nbsp;...&nbsp;,&nbsp;n。请你求出全部页码中所有单个数字的和，例如第123页，它的和就是1+2+3=6。 

 
 # 输入格式 
一行为n(1&nbsp;&lt;=&nbsp;n&nbsp;&lt;=&nbsp;10^9)。 

 
 # 输出格式 
一行，代表所有单个数字的和。
