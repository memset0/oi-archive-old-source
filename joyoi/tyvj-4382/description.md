# 

 
 # 题目背景 
<p>小明要走一个N级的楼梯。</p> 

 
 # 题目描述 
<p>一次可以走a1,a2,a3...am级，问最少走几次。</p> 

 
 # 输入格式 
<p>第一行两个整数N，M</p>

<p>第二行M个整数表示a1,a2...am</p>

<p>样例输入：</p>

<p>3&nbsp;2</p>

<p>1&nbsp;2</p> 

 
 # 输出格式 
<p>一个整数表示最少走路次数</p>

<p>样例输出（对于上面的输入）：</p>

<p>2</p> 

 
 # 提示 
<p>0&lt;=a1...am&lt;=100</p>

<p>1&lt;=m&lt;=1000000</p>

<p>1&lt;=n&lt;=1000000</p>

<p>无法走到第N级时输出-1</p>

<p>你可以走到第N+1,N+2&hellip;&hellip;层</p>
