 # 题目描述 
<p>
N个矩形,排成一排.<br>现在希望用尽量少的矩形海报Cover住它们.</p> 

 
 # 输入格式 
<p>
第一行给出数字N,代表有N个矩形.N在[1,250000]<br>下面N行,每行给出矩形的长与宽.其值在[1,1000000000]2 1/2 Postering</p> 

 
 # 输出格式 
<p>
最少数量的海报数.</p>
