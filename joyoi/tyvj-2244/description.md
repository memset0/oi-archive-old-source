 # 题目描述 
<p>
求3<=(3*x) Mod 5<=4的最小正整数解<br></p> 

 
 # 输入格式 
<p>
第一行一个整数。T代表数据组数。<br>接下来T行，每行4个数，M，D，L，R表示L<=(D*x Mod M)<=R。<br></p> 

 
 # 输出格式 
<p>
共T行，表示x的最小正整数解，若无解则输出-1。<br><br></p> 

 
 # 提示 
<p>
T<=100<br>1<=M、D、L、R<=1000000000<br></p>
