 # 题目描述 
<p>
　　设有n个正整数（n≤20），将它们联接成一排，组成一个最大的多位整数。例如：n=3时，3个整数13，312，343联接成的最大整数为：34331213。又如：n=4时，4个整数7，13，4，246联接成的最大整数为：7424613<br><br></p> 

 
 # 输入格式 
<p>
n<br>n个数<br><br>（注：n小于等于20，每个数值都在longint之内）<br></p> 

 
 # 输出格式 
<p>
联接成的多位数  <br></p>
