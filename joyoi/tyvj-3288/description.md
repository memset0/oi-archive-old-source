 # 题目描述 
<p>
最小伤害<br><br>问题描述：<br>　　把儿站在一个N x N的方阵中最左上角的格子里。他可以从一个格子走到它右边和下边的格子里。每一个格子都有一个伤害值。他想在受伤害最小的情况下走到方阵的最右下角。<br></p> 

 
 # 输入格式 
<p>
　　第一行输入一个正整数n。<br>　　以下n行描述该矩阵。矩阵中的数保证是不超过1000的正整数。<br></p> 

 
 # 输出格式 
<p>
　　输出最小伤害值。<br><br></p> 

 
 # 提示 
<p>
数据规模：<br>    n <= 1000<br><br></p>
