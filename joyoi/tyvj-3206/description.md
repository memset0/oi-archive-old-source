 # 题目描述 
<p>
　　输入一个分数的分子和分母，输出它的小数形式，精确到小数点后20位。若有循环节，请标明，如：1/3=0.(3),如果结果是一个整数，则直接输出该整数即可，如：分子为64,分母为8,则输出结果为：8。 </p> 

 
 # 输入格式 
<p>
　　一行，两个整数，中间有一空格隔开，分别为分子、分母。</p> 

 
 # 输出格式 
<p>
　　一行，小数表示形式。</p>
