 # 题目描述 
<p>解一元N次方程。</p>

<p>每个系数已化到整数（2^30以内），最简。</p>

<p>等式右边为0.</p>

<p>输出所有的实数根即可。</p>

<p>保留6位小数。</p>

<p>字母为小写的x</p>

<p>&nbsp;</p>

<p>样例输入：</p>

<p>x^2-3x+2=0</p>

<p>样例输出：</p>

<p>x1=1</p>

<p>x2=2</p>

<p>&nbsp;</p> 

 
 # 输入格式 
<p>一行。输入方程。</p> 

 
 # 输出格式 
<p>x1=...</p>

<p>x2=...</p>

<p>...</p>

<p>注意：输出的x1&nbsp;x2&nbsp;...&nbsp;xn各不相同，且从小到大输出。</p>
