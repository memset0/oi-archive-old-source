# 题目描述
给你一个数字ID，请你输出这个物品名称。
# 输入格式
一行一个整数，代表数字ID。
# 输出格式
一行一个字符串，格式是：9个英文字母+英文半角冒号+物品名称。
# 提示与说明
数字ID$\leq$50

物品附加参数为0
