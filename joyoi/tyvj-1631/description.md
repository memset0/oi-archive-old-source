 # 题目描述 
定义jb数为：一个数分解质因数后，所有质因子的和。例如jb(4)=4,jb(5)=5，jb(6)=5等。现给出一个范围，求出这个范围内jb数的和。由于这个数可能过大，只需要求出这个数模1919的值就可以了。 

 
 # 输入格式 
第一行为T，为T组测试数据。<BR>后T行每行两个数A,B，为一个范围。 

 
 # 输出格式 
共T行，每行输出一个和。 

 
 # 提示 
数据范围：T&lt;=10，2&lt;=A&lt;=B&lt;=10万
