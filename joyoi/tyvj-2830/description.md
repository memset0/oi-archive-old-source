 # 题目描述 
<p>
　　栈是常用的一种数据结构，有n令元素在栈顶端一侧等待进栈，栈顶端另一侧是出栈序列。你已经知道栈的操作有两&#8226;种：push和pop，前者是将一个元素进栈，后者是将栈顶元素弹出。现在要使用这两种操作，由一个操作序列可以得到一系列的输出序列。请你编程求出对于给定的n，计算并输出由操作数序列1，2，…，n，经过一系列操作可能得到的输出序列总数。<br><br><center><img src="/source/joyoi/tyvj-2830/img/aHR0cDovL3d3dy5qb3lvaS5jbi9wcm9ibGVtL3R5dmotMjgzMC9wcm9ibGVtc19pbWFnZXMvMTIwMS8xLmpwZw==.jpg"></img></center></p> 

 
 # 输入格式 
<p>
　　一个整数n（1 <= n <= 15）</p> 

 
 # 输出格式 
<p>
　　一个整数，即可能输出序列的总数目。</p>
