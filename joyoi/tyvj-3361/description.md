 # 题目描述 
<p>
　　求n!的尾零个数，即从最低位开始向高位数，至第一个非零位停止，零的个数。</p> 

 
 # 输入格式 
<p>
　　一行，一个数n。（n <= 10^6 ）</p> 

 
 # 输出格式 
<p>
　　一行，一个整数值，尾零的个数。</p>
