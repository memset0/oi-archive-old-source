 # 题目描述 
<p>
windy定义了一种windy数。<br>不含前导零且相邻两个数字之差至少为2的正整数被称为windy数。<br>windy想知道，在A和B之间，包括A和B，总共有多少个windy数？<br><br></p> 

 
 # 输入格式 
<p>
包含两个整数，A B。<br><br></p> 

 
 # 输出格式 
<p>
一个整数。<br><br></p>
