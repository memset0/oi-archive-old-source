 # 题目描述 
定义最大奇因数——N（x）表示x的最大奇因数，例如N(1)=1，N(2)=1,N(5)=5,N(11)=11，N（12）=3,等等，S(X)=N(1)+N(2)+N(3)+.....+N(2^X);<BR> 

 
 # 输入格式 
输入一行，一个自然数n，（n&lt;=30） 

 
 # 输出格式 
输出一行，一个自然数S（n）;<BR> 

 
 # 提示 
小心数据范围
