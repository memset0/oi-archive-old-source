 # 题目描述 
在N个数中找出其和为M的若干个数。先读入正整数N（1&lt;N&lt;100）和M(1&lt;M&lt;10000)，&nbsp;再读入N个正数（可以有相同的数字，每个数字均在1000以内），&nbsp;在这N个数中找出若干个数，&nbsp;使它们的和是M，&nbsp;把满足条件的数字组合都找出来以统计组合的个数，输出组合的个数（不考虑组合是否相同）。要求你的程序运行时间不超过1秒。 

 
 # 输入格式 
第一行是两个数字，表示N和M。<BR>第二行起是N个数。<BR> 

 
 # 输出格式 
就一个数字，表示和为M的组合的个数。
