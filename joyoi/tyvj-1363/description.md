 # 题目描述 
一列火车n节车厢，依次编号为1,2,3,…,n。每节车厢有两种运动方式，进栈与出栈，问n节车厢出栈的可能排列方式有多少种。 

 
 # 输入格式 
一个数，n(n&lt;=30000) 

 
 # 输出格式 
一个数s表示n节车厢出栈的可能排列方式 

 
 # 提示 
来自邵铮论文
