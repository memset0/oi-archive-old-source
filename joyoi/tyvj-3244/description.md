 # 题目描述 
<p>
判断点在线段上（linein.pas/c/cpp）<br><br>【问题描述】<br><br>判断二维平面上的一个点是否在一条线段上。</p> 

 
 # 输入格式 
<p>
输入两行<br>第一行两个数，表示点在平面上的x，y坐标值<br>第二行四个数，表示在平面上的线段的两个端点的坐标值(x1,y1),(x2,y2).</p> 

 
 # 输出格式 
<p>
如果点在线段上，输出“YES”<br>否则输出“NO”</p>
