 # 题目描述 
<p>
　　有一排硬币，共n个，全部是正面向上，一次可以翻其中任意的m个，这m个不要求是连续排列的，但一次必须翻m个，现在问：最少翻几次，可以使得硬币全面正面向下？</p> 

 
 # 输入格式 
<p>
　　一行，两个整数，分别是n和m，中间一个空格间隔（m <= n ， n <= 50 ）。</p> 

 
 # 输出格式 
<p>
　　一行，一个整数，即最少的翻转次数，如果无法达到目标，则输出数值：-1。</p>
