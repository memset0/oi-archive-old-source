 # 题目描述 
输入一个长度为n的整数序列，从中找出一段不超过M的连续子序列，使得整个序列的和最大。<BR><BR>例如&nbsp;1，-3,5,1，-2,3<BR><BR>当m=4时，S=5+1-2+3=7<BR>当m=2或m=3时，S=5+1=6 

 
 # 输入格式 
第一行两个数n,m<BR>第二行有n个数，要求在n个数找到最大子序和 

 
 # 输出格式 
一个数，数出他们的最大子序和 

 
 # 提示 
数据范围：<BR>100%满足n,m&lt;=300000
