# 

 
 # 题目背景 
<p>10月3日是星星的生日，大家在一起吃蛋糕</p> 

 
 # 题目描述 
<p>星星买了一块N米*M米的蛋糕，正好有N*M个人，所以每人一块1米*1米的蛋糕，问最少要几刀才能把整块蛋糕切完？</p> 

 
 # 输入格式 
<p>两个正整数N，M<br />
(1&lt;=N,M&lt;=10000)</p> 

 
 # 输出格式 
<p>一个数，最少要切几刀</p> 

 
 # 提示 
<h1>测试样例</h1>

<h3>输入</h3>

<blockquote>
<p>2&nbsp;2</p>
</blockquote>

<h3>输出</h3>

<blockquote>
<p>3</p>
</blockquote>
