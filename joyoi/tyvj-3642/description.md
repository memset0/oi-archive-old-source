 # 输入格式 
<p>
第一行为整数L，其中4<=L<=100000，且有50%的数据满足L<=104，表示木板下侧直线段的长。第二行为L个正整数A1,A2,…,AL，其中1<Ai<=108，表示第i段的高度。<br></p> 

 
 # 输出格式 
<p>
仅包含一个整数D，表示为使梳子面积最大，需要从木板上挖掉的格子数。<br></p>
