 # 题目背景 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">NOIP2014普及组第一题</span></p> 

 
 # 题目描述 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">珠心算是一种通过在脑中模拟算盘变化来完成快速运算的一种计算技术。珠心算训练，</span><br style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;" />
<span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">既能够开发智力，又能够为日常生活带来很多便利，因而在很多学校得到普及。</span><br style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;" />
<span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">&nbsp;&nbsp;某学校的珠心算老师采用一种快速考察珠心算加法能力的测验方法。他随机生成一个正</span><br style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;" />
<span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">整数集合，集合中的数各不相同，然后要求学生回答：其中有多少个数，恰好等于集合中另</span><br style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;" />
<span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">外两个（不同的）数之和？</span><br style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;" />
<span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">&nbsp;&nbsp;最近老师出了一些测验题，请你帮忙求出答案。</span></p> 

 
 # 输入格式 
<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">输入共两行，第一行包含一个整数&nbsp;n，表示测试题中给出的正整数个数。<br />
&nbsp;&nbsp;第二行有&nbsp;n&nbsp;个正整数，&nbsp;每两个正整数之间用一个空格隔开，表示测试题中给出的正整数。</p>

<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">样例：</p>

<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">4</p>

<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">1&nbsp;2&nbsp;3&nbsp;4</p> 

 
 # 输出格式 
<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">输出共一行，包含一个整数，表示测验题答案。</p>

<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">样例：</p>

<p style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px; line-height: 20px;">2</p>
