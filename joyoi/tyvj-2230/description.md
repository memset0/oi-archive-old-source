 # 题目描述 
<p>
给出1~n的一个排列，统计该排列有多少个长度为奇数的连续子序列的中位数是b。中位数是指把所有元素从小到大排列后，位于中间的数。<br></p> 

 
 # 输入格式 
<p>
第一行为两个正整数n和b ，第二行为1~n 的排列。<br></p> 

 
 # 输出格式 
<p>
输出一个整数，即中位数为b的连续子序列个数。</p> 

 
 # 提示 
<p>
第三个样例解释：{4}, {7,2,4}, {5,7,2,4,3}和{5,7,2,4,3,1,6}<br>N<=100000</p>
