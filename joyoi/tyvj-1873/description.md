 # 题目描述 
设有n个正整数（n≤20），将它们联接成一排，组成一个最大的多位整数。<BR>例如：n=3时，3个整数13，312，343联接成的最大整数为：34331213<BR>又如：n=4时，4个整数7，13，4，246联接成的最大整数为：7424613<BR> 

 
 # 输入格式 
第一行n<BR>第二行n个数,第个数用空隔隔开。<BR> 

 
 # 输出格式 
一个数，联接成的多位数<BR>
