 # 题目描述 
<p>
给出一个N*N的数字矩阵.<br>你从左上角出发，可以向右走，或者向下走，不能出界，你所移动的长度为当前你所在的格子的数字的值.</p> 

 
 # 输入格式 
<p>
先给出一个数字N,N<=100<br>再给出N*N的数字矩阵,每个数字的值在10以内</p> 

 
 # 输出格式 
<p>
从左上角走到右下角有多少种走法.</p>
