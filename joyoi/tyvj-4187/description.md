# 

 
 # 题目背景 
<p>一天，星星看着天上的星星，突然发现有一些星星围成了一个圈，并且隔一秒就会掉下一颗星</p> 

 
 # 题目描述 
<p>假如有N颗星，并且把其中一颗星星定为1号星，则从1号星开始数1，每数到M，那颗星便会掉下来，接着又从1开始数&hellip;&hellip;问最后留下的是几号星？（赤裸裸的约瑟夫）</p> 

 
 # 输入格式 
<p>两个正整数N，M<br />
(1&lt;=N,M&lt;=1000000)</p> 

 
 # 输出格式 
<p>一个数，最后留下的星的编号</p> 

 
 # 提示 
<h1>测试样例</h1>

<h3>输入</h3>

<blockquote>
<p>6&nbsp;2</p>
</blockquote>

<h3>输出</h3>

<blockquote>
<p>5</p>
</blockquote>
