 # 题目描述 
<p>
N柱砖,希望有连续K柱的高度是一样的.<br>你可以选择以下两个动作<br>1:从某柱砖的顶端拿一块砖出来,丢掉不要了.<br>2:从仓库中拿出一块砖,放到另一柱.仓库无限大.<br>现在希望用最小次数的动作完成任务.</p> 

 
 # 输入格式 
<p>
第一行给出N,K. (1 ≤ k ≤ n ≤ 100000),<br>下面N行,每行代表这柱砖的高度.0 ≤ hi ≤ 1000000</p> 

 
 # 输出格式 
<p>
最小的动作次数</p> 

 
 # 提示 
<p>
原题还要求输出结束状态时,每柱砖的高度.本题略去.</p>
