 # 题目描述 
<p>
致力于建设全国示范和谐小村庄的H村村长dadzhi，决定在村中建立一个瞭望塔，以此加强村中的治安。<br>我们将H村抽象为一维的轮廓。如下图所示<br> <br>我们可以用一条山的上方轮廓折线(x1, y1), (x2, y2), …. (xn, yn)来描述H村的形状，这里x1 < x2 < …< xn。瞭望塔可以建造在[x1, xn]间的任意位置, 但必须满足从瞭望塔的顶端可以看到H村的任意位置。可见在不同的位置建造瞭望塔，所需要建造的高度是不同的。为了节省开支，dadzhi村长希望建造的塔高度尽可能小。<br>请你写一个程序，帮助dadzhi村长计算塔的最小高度。<br></p> 

 
 # 输入格式 
<p>
第一行包含一个整数n，表示轮廓折线的节点数目。接下来第一行n个整数, 为x1 ~ xn. 第三行n个整数，为y1 ~ yn。<br></p> 

 
 # 输出格式 
<p>
仅包含一个实数，为塔的最小高度，精确到小数点后三位。<br></p>
