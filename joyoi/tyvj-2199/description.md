 # 输入格式 
<p>
　　输入文件的第一行是一个正整数n(1≤n≤1000)，表示多米诺骨牌数。接下来的n行表示n个多米诺骨牌的点数。每行有两个用空格隔开的正整数，表示多米诺骨牌上下方块中的点数a和b，且1≤a，b≤6。</p> 

 
 # 输出格式 
<p>
　　输出文件仅一行，包含一个整数。表示求得的最小旋转次数。</p>
