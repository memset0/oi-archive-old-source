 # 题目描述 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px;">给出三个线段的长度，请写一个程序，判断能否构成一个三角形？</span></p> 

 
 # 输入格式 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px;">输入只有一行，包含三个正整数，用空格隔开。</span></p> 

 
 # 输出格式 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px;">若能构成三角形，输出&ldquo;YES&rdquo;，反之输出&ldquo;NO&rdquo;，不含引号。</span></p> 

 
 # 提示 
<p><span style="color: rgb(0, 0, 0); font-family: 'Microsoft YaHei UI', 'Microsoft YaHei', 'Segou UI'; font-size: 14px;">每个输入的正整数都小于等于10000。</span></p>
