 # 题目描述 
<p>已知某些特定的01串是病毒的特征代码。如果一个01串不含有任何病毒特征代码，则称它为一段安全代码。给定病毒特征库，判断是否存在无限长的安全代码。</p> 

 
 # 输入格式 
<p>第一行为一个整数n，表示病毒特征代码的条数。下面n行，每行一段病毒特征代码。所有代码长度之和不超过30000。</p> 

 
 # 输出格式 
<p>若存在无限长的安全代码，输出一行&ldquo;TAK&rdquo;，否则输出一行&ldquo;NIE&rdquo;。</p>
