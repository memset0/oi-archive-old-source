 # 题目描述 
给出一个数列，求其中最长的波动序列<BR>波动序列：<BR>a1&gt;a2&lt;a3&gt;a4......<BR>或a1&lt;a2&gt;a3&lt;a4...... 

 
 # 输入格式 
第一行，一个数N<BR>第二行，N个数，表示这个序列<BR> 

 
 # 输出格式 
一个数，表示最长波动序列的长度。<BR> 

 
 # 提示 
喜欢挑战的同学可以试试不开数组<BR>40%&nbsp;n&lt;=100<BR>70%&nbsp;n&lt;=10000<BR>100%&nbsp;n&lt;=1000000
