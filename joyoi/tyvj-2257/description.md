 # 题目描述 
<p>
给出平面上N个点,N<=10^5.请求出一个半径最小的圆覆盖住所有的点<br></p> 

 
 # 输入格式 
<p>
第一行给出数字N，现在N行，每行两个实数x,y表示其坐标.<br><br></p> 

 
 # 输出格式 
<p>
输出最小半径，输出保留三位小数.<br></p>
