 # 题目描述 
有n个石子围成一圈，每个石子都有一个权值a[i]，你需要取一些石子，每个石子的得分是a[i]*d，d表示这个石子到两边被取了的石子的距离和。<BR>现在你可以取若干石子，使得分最大。<BR> 

 
 # 输入格式 
第1行一个整数n。<BR>接下来n行，每行一个整数a[i]。<BR> 

 
 # 输出格式 
仅一个整数，表示最大得分。 

 
 # 提示 
1≤a[i]≤100000<BR>对于30%的数据，n≤60<BR>对于60%的数据，n≤300<BR>对于100%的数据，n≤100000<BR>
