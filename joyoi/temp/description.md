# 题目背景
gcd(a, b) 表示 a 和 b 的[最大公约数](https://baike.baidu.com/item/%E6%9C%80%E5%A4%A7%E5%85%AC%E7%BA%A6%E6%95%B0).
# 题目描述
给你两个数 a 和 b, 求 gcd(a,b).
# 数据范围
a, b <= 10^9# 样例数据
