 # 题目描述 
有价值分别为1..6的大理石各a[1..6]块，现要将它们分成两部分，使得两部分价值之和相等，问是否可以实现。其中大理石的总数不超过20000。&nbsp; 

 
 # 输入格式 
有多组数据！<BR>所以可能有多行<BR>如果有0&nbsp;0&nbsp;0&nbsp;0&nbsp;0&nbsp;0表示输入文件结束<BR>其余的行为6个整数 

 
 # 输出格式 
有多少行可行数据就有几行输出<BR>如果划分成功，输出Can，否则Can't
