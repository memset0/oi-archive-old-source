 # 题目描述 
<p>有一个箱子容量为V（正整数，0＜＝V＜＝20000），同时有n个物品(0＜n＜＝30)，每个物品有一个体积（正整数）。<br />
<br />
要求n个物品中，任取若干个装入箱内，使箱子的剩余空间为最小。</p> 

 
 # 输入格式 
<p>第一行,一个整数，表示箱子容量<br />
第二行,一个整数，表示有n个物品<br />
接下来n行，分别表示这n&nbsp;个物品的各自体积</p> 

 
 # 输出格式 
<p>一个整数，表示箱子剩余空间。</p>
