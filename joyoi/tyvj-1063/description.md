 # 题目描述 
给你一个长度为n的数字串，数字串里会包含1-m这些数字。如果连续的一段数字子串包含了1-m这些数字，则称这个数字字串为NUM串。你的任务是求出长度最短的NUM串是什么，只需要输出这个长度即可。<BR>1&lt;=n，m&lt;=200000 

 
 # 输入格式 
第一行给定n和m。&nbsp;<BR>第二行n个数，表示数字串，数字间用空格隔开。<BR> 

 
 # 输出格式 
如果存在NUM串则输出最短NUM串长度，否则输出“NO”。
