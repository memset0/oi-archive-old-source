 # 输入格式 
<p>输入一串由字母，字母顺序，ascll和dz码组成的密码。输出组成的字母。</p> 

 
 # 输出格式 
<p>输出组成的字母。</p> 

 
 # 提示 
<p>1、{题目备注}</p>

<p>输入中每个字符、码值之间都要空一个空格。</p>

<p>输出中不需要空格。</p>

<p>字母顺序中有重复，所以在小写字母顺序后面要加&ldquo;~&rdquo;号。</p>
