 # 题目描述 
<p>
硬币购物<br>一共有4种硬币。面值分别为c1,c2,c3,c4。某人去商店买东西，去了tot次。<br>每次带di枚ci硬币，买si的价值的东西。请问每次有多少种付款方法。<br></p> 

 
 # 输入格式 
<p>
第一行<br>c1,c2,c3,c4,tot<br>下面tot行<br>d1,d2,d3,d4,s<br></p> 

 
 # 输出格式 
<p>
每次的方法数<br></p>
