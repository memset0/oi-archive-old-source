 # 题目描述 
<p>
一个n*n(1<=n<=100)的棋盘上放置n个国际象棋的皇后，使其不能相互攻击，即任何两个皇后都不能处于棋盘上的同一行、同一列、同一条斜对角线上，试问功有多少中摆法？<br></p> 

 
 # 输入格式 
<p>
n（N〈=12）<br></p> 

 
 # 输出格式 
<p>
一行，所有的方案。</p>
