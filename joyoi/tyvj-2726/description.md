 # 输入格式 
<p>
输入共一行，10个正整数，依次为<br>C,N,xzero,xprod,xadd,xmod,yzero,yprod,yadd,ymod。 </p> 

 
 # 输出格式 
<p>
输出共一行，为所求的最小值。 </p> 

 
 # 提示 
<p>
对于全部的数据，1<=C<=16，1<=N<=10000，1<=xmod,ymod<=1000000000，<br>0<=xzero,xprod,xadd<xmod，0<=yzero,yprod,yadd<ymod。 <br> </p>
