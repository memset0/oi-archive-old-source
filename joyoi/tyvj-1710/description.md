 # 题目描述 
给你一个数N，求出最小的B（B&gt;=2），使得N在B进制下为回文数。 

 
 # 输入格式 
第一行1&nbsp;个整数TEST，表示数据组数。<BR>接下来TEST&nbsp;行，每行一个整数N。 

 
 # 输出格式 
共输出TEST&nbsp;行，每行对应一个答案B。 

 
 # 提示 
30%的数据&nbsp;TEST&lt;=10，N&lt;=10^4。<BR>100%的数据&nbsp;TEST&lt;=1000，N&lt;=10^10。
