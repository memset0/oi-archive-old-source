 # 题目描述 
<p>
有N头奶牛，每头那牛都有一个标号Pi，1 <= Pi <= M <= N <= 40000。<br>现在Farmer John要把这些奶牛分成若干段，定义每段的不河蟹度为：若这段里有k个不同的数，那不河蟹度为k*k。那总的不河蟹度就是所有段的不河蟹度的总和。<br></p> 

 
 # 输入格式 
<p>
第一行：两个整数N，M<br>第2..N+1行：N个整数代表每个奶牛的编号<br></p> 

 
 # 输出格式 
<p>
一个整数，代表最小不河蟹度<br></p>
