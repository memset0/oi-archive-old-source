 # 输入格式 
<p>
输入仅包含一个数字n，n一定会是偶数，因为会有相同片段数量的上坡以及下坡。</p> 

 
 # 输出格式 
<p>
请输出山顶位置由左而右非递减的山稜线形状总数。<br>由于答案可能很大，你只要输出以十进位表示时，它的最后9位数即可。<br></p> 

 
 # 提示 
<p>
提示 ：<br>佔总分20%的测试数据中 n<=60<br>佔总分40%的测试数据中 n<=200<br>佔总分100%的测试数据中 n<=3000<br></p>
