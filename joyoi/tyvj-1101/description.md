 # 题目描述 
按照字典序将字符串数组排序 

 
 # 输入格式 
第1行n&nbsp;表示n个字符串<BR>第2到n+1行为n个字符串&nbsp;只会出现小写字母 

 
 # 输出格式 
N行，每行分别一个字符串，按照字典序顺序 

 
 # 提示 
n&lt;=10000<BR>每个字符串长度不超过256
