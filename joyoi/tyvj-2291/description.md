 # 题目描述 
<p>
给出N个矩形,N<=10000.其坐标不超过10^9.求其面积并</p> 

 
 # 输入格式 
<p>
先给出一个数字N,代表有N个矩形.<br>接下来N行,每行四个数,代表矩形的坐标.</p> 

 
 # 输出格式 
<p>
输出面积并</p>
