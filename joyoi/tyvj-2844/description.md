 # 题目描述 
<p>
有一个无穷序列如下： <br>110100100010000100000…… <br>请你找出这个无穷序列中指定位置上的数字<br></p> 

 
 # 输入格式 
<p>
第一行一个正整数N，表示询问次数。<br>接下来的N行每行一个正整数Ai，Ai表示在序列中的位置。<br></p> 

 
 # 输出格式 
<p>
N行，每行为0或1，表示序列第Ai位上的数字。</p> 

 
 # 提示 
<p>
【数据范围】<br>对于100%的数据有N<=1500000，Ai<=10^9<br><br></p>
