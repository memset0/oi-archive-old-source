 # 题目描述 
简单的计算几何<BR>在平面直角坐标系中有三个点A，B，C<BR>求三角形ABC中角BAC的平分线与BC的交点D的坐标<BR> 

 
 # 输入格式 
三行，每行两个整数，表示A，B，C的坐标<BR> 

 
 # 输出格式 
一行两个数用空格隔开，表示D的坐标，保留两位小数<BR> 

 
 # 提示 
所有坐标均在[-10000,10000]之内
