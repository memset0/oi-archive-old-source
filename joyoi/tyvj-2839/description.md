 # 输入格式 
<p>
输入文件flip.in包含4行，每行4个字符，每个字符"w" 或 "b"表示游戏开始时格子上物件的状态。</p> 

 
 # 输出格式 
<p>
输出文件flip.out仅一个整数，即从给定状态到实现这一任务的最少翻转次数。如果给定的状态就已经实现了目标就输出0，如果不可能实现目标就输出"Impossible"。</p>
