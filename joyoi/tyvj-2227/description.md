 # 题目描述 
<p>
TBL和X用巧克力棒玩游戏。每次一人可以从盒子里取出若干条巧克力棒，或是将一根取出的巧克力棒吃掉正整数长度。TBL先手<br>两人轮流，无法操作的人输。 他们以最佳策略一共进行了10轮（每次一盒）。你能预测胜负吗？</p> 

 
 # 输入格式 
<p>
输入数据共20行。 第2i-1行一个正整数Ni，表示第i轮巧克力棒的数目。 第2i行Ni个正整数Li,j，表示第i轮巧克力棒的长度。</p> 

 
 # 输出格式 
<p>
输出数据共10行。 每行输出“YES”或“NO”，表示TBL是否会赢。<br>如果胜则输出"NO",否则输出"YES"</p> 

 
 # 提示 
<p>
YES<br>NO<br>其它8组输出&#8222;&#8222;）</p>
