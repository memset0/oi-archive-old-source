 # 题目描述 
<p>
平面上有若干个点，现在要求用最少的底边在X轴上且面积小等A的矩形覆盖所有点，这些矩形可以重叠。<br>N<=100，A<=2000000<br></p> 

 
 # 输入格式 
<p>
N,A<br>以下N行输入点坐标。</p> 

 
 # 输出格式 
<p>
</p> 

 
 # 提示 
<p>
1 ≤ N ≤ 100 <br>1 ≤ A ≤ 200 000 <br>Each point has 0 ≤ x ≤ 3 000 000 and 1 ≤ y ≤ A <br>For 30% of the test cases, 1 ≤ N ≤ 18 </p>
