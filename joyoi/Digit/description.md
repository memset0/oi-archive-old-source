# 问题描述
给出一个不多于5位的正整数，要求：
1.求出它是几位数
2.分别输出每一位数字
3.逆序输出各位数字
## 输入
给出一个不多于5位的正整数
## 输出
1.求出它是几位数
2.分别输出每一位数字
3.逆序输出各位数字
