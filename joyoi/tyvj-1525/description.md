 # 题目背景 
化学方程式合并之后……<BR>魂之挽歌又纠结于化学反应热的计算…… 

 
 # 题目描述 
现在需要你来计算化学反应热。<BR>给出两个方程式及他们的反应热，<BR>计算目标方程式的反应热。<BR>保证三个方程式中的物质均为A，B，C，D<BR><BR> 

 
 # 输入格式 
前两行，组成如下：<BR>1、一个方程式<BR>2、它的反应热<BR>第三行，目标方程式。 

 
 # 输出格式 
目标方程式的反应热
