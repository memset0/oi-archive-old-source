 # 题目描述 
<p>
	三维凸包问题是一个基础的三维计算几何问题，不过这次你只需要做一个简单版的三维凸包问题就行了。<br><br></p> 

 
 # 输入格式 
<p>
	输入数据一共有若干行，每行三个整数，表示一个点的坐标。点的个数为五个。<br><br></p> 

 
 # 输出格式 
<p>
	输出一个实数，保留两位小数，表示三维凸包的体积。<br><br></p> 

 
 # 提示 
<p>
	对于所有数据，坐标范围在[0,100]*[0,100]*[0,100]。<br></p>
