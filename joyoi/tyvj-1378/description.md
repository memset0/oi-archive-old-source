 # 题目背景 
为了方便beginners,特设此题，为大家增加RP。^_^ 

 
 # 题目描述 
给你两个数a和b，求出a+b的值。 

 
 # 输入格式 
共两行，第一行为数&nbsp;a，<BR>第二行为数b。<BR> 

 
 # 输出格式 
&nbsp;&nbsp;&nbsp;&nbsp;一个数，即a+b的值 

 
 # 提示 
保证所有a,b范围&lt;=maxlongint;<BR>没说下界，允许小数和分数，你们懂得。。。。。<BR>对于小数，数据保证长度不超过256...(都这么写)<BR>对于分数，a和b分子分母不超过maxlongint，能通分时要通分。<BR>
