 # 输入格式 
<p>
第1行为一个整数N(1<=N<=15)，即野人的数目。第2行到第N+1每行为三个整数Ci, Pi, Li (1<=Ci,Pi<=100, 0<=Li<=106 )，表示每个野人所住的初始洞穴编号，每年走过的洞穴数及寿命值。<br><br></p> 

 
 # 输出格式 
<p>
仅包含一个数M，即最少可能的山洞数。输入数据保证有解，且M不大于106。<br><br><br></p>
