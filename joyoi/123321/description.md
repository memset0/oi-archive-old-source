# 
【试题描述】某人想把 N 元钱换成 10 元、5 元、1 元这样的零钱， 在这三种零钱中每种零钱都至少各有一张的情况下，共有多少种兑换方案。 
【输入格式】 
输入由文件 exchange.in 提供。只有一个整数 N（16<=N<=10^5）。 
【输出格式】输出到文件 exchange.out。输出兑换方案的总数。 
【样例输入】 
100 
【样例输出】 
81
