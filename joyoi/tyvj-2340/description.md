 # 输入格式 
<p>
输入数据首先输入两个整数N,M，表示了迷宫的边长。<br>接下来N行，每行M个字符，描述了迷宫。<br></p> 

 
 # 输出格式 
<p>
若小AA能够赢得游戏，则输出一行"WIN"，然后输出所有可以赢得游戏的起始位置，按行优先顺序输出<br>每行一个,否则输出一行"LOSE"（不包含引号）。</p> 

 
 # 提示 
<p>
对于100%的数据，有1≤n,m≤100。 <br>对于30%的数据，有1≤n,m≤5。</p>
