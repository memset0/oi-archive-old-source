 # 题目描述 
给定一个数列每项是1或者2.<br>问是否存在一个子段。和是指定的数。。<br><br>数列长度&lt;=20w&nbsp;询问数&lt;=20w&nbsp;询问的数字在int范围内。<br><br> 

 
 # 输入格式 
第一行：数列长度N<br>第二行：N个数,每个数不是1就是2，中间木有空格隔开<br>第三行：询问数M<br>第四行：M个询问，中间有空格隔开<br><br> 

 
 # 输出格式 
每个询问一行，YES或者NO。<br><br>
