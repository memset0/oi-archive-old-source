 # 题目描述 
<p>
核电站问题<br>　　一个核电站有N个放核物质的坑，坑排列在一条直线上。如果连续M个坑中放入核物质，则会发生爆炸，于是，在某些坑中可能不放核物质。<br>　　任务：对于给定的N和M，求不发生爆炸的放置核物质的方案总数。</p> 

 
 # 输入格式 
<p>
　　输入文件只一行，两个正整数N，M( 1 < N < 50，2 ≤ M ≤ 5 )<br></p> 

 
 # 输出格式 
<p>
　　输出文件只有一个正整数S，表示方案总数。</p>
