 # 题目描述 
<p>
有n个圆盘从天而降，后面落下的可以盖住前面的。求最后形成的封闭区域的周长。<br></p> 

 
 # 输入格式 
<p>
n<br>ri xi y1<br>...<br>rn xn yn<br></p> 

 
 # 输出格式 
<p>
最后的周长，保留三位小数<br></p>
