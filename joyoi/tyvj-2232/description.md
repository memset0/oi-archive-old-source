 # 输入格式 
<p>
第一行包含一个正整数n，队伍的个数。第二行包含n个非负整数，即每支队伍的得分。<br></p> 

 
 # 输出格式 
<p>
输出仅一行，即可能的分数表数目。保证至少存在一个可能的分数表。<br></p> 

 
 # 提示 
<p>
N<=8</p>
