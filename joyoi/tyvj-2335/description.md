 # 题目描述 
<p>
回文串的定义就是一个字符串C从前往后读和从后往前读一样。<br>现在给你一个字符串T，每次可以交换2个相邻的字符，你的目标就是通过尽可能少的次数使T变成回文串。<br>如果无解，输出“Impossible!”<br><br></p> 

 
 # 输入格式 
<p>
第1行一个数m，表示有m组数据。<br>下面m行，每行首先是一个数n，表示这一行的字符的长度，接着是一个长为n的字符串。<br><br></p> 

 
 # 输出格式 
<p>
m行，每行一个数，表示最小交换次数。<br><br></p>
