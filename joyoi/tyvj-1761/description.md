 # 题目描述 
同一时刻有N位车主带着他们的爱车来到了汽车维修中心。维修中心共有M位技术人员，不同的技术人员对不同的车进行维修所用的时间是不同的。现在需要安排这M位技术人员所维修的车及顺序，使得顾客平均等待的时间最小。<BR>说明：顾客的等待时间是指从他把车送至维修中心到维修完毕所用的时间。<BR> 

 
 # 输入格式 
第一行有两个数M,N，表示技术人员数与顾客数。<BR>接下来n行，每行m个整数。第i+1行第j个数表示第j位技术人员维修第i辆车需要用的时间T。<BR> 

 
 # 输出格式 
最小平均等待时间，答案精确到小数点后2位。 

 
 # 提示 
(2&lt;=M&lt;=9,1&lt;=N&lt;=60),&nbsp;(1&lt;=T&lt;=1000)
