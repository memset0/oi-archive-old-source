# 

 
 # 题目背景 
<p>开学了，星星和他的小伙伴们在一起做游戏，这时，阳阳想出了一个游戏</p> 

 
 # 题目描述 
<p>有N个数，你可以从中取出两个数，再将它们的和放回去，直到只有一个数。总分为每一次操作的和的总和，总分越小名次越高，星星想得第一，可惜他此时没有电脑，他打电话给你，你能帮帮他吗？</p> 

 
 # 输入格式 
<p>第一行一个整数N。(1&lt;=N&lt;=5000)<br />
第二行N个整数。</p> 

 
 # 输出格式 
<p>一个数，为最小的总分（保证结果不超过32位整数）</p> 

 
 # 提示 
<h1>测试样例</h1>

<h3>输入</h3>

<blockquote>
<p>3<br />
1&nbsp;2&nbsp;3</p>
</blockquote>

<h3>输出</h3>

<blockquote>
<p>9</p>
</blockquote>
