 # 题目描述 
<p>
监狱有连续编号为1...N的N个房间，每个房间关押一个犯人，有M种宗教，每个犯人可能信仰其中一种。如果相信房间的犯人的宗教相同，就可能发生越狱，求有多少种状态可能发生越狱<br></p> 

 
 # 输入格式 
<p>
输入两个整数M，N.1<=M<=10^8,1<=N<=10^12<br></p> 

 
 # 输出格式 
<p>
可能越狱的状态数，模100003取余<br></p> 

 
 # 提示 
<p>
<br>6种状态为(000)(001)(011)(100)(110)(111)</p>
