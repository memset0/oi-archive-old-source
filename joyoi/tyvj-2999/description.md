 # 输入格式 
<p>
第一行是三角阵的等级Ｎ（Ｎ≤30）。<br>第二行和第三行都是一个长度为N的字符串，由大写字母“L”、“R”、“T”组成，表示两个小三角形的标号。<br></p> 

 
 # 输出格式 
<p>
输出一个数，表示从一个小三角形走到另一个小三角形所需的最少步数。</p> 

 
 # 提示 
<p>
从“TRL”出发，途经“TRR”、“RTT”、“RTL”、“RLT”，最后到达“RLR”，一共需要5步。</p>
