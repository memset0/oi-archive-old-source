 # 题目描述 
<p>
给定一个羊圈，其为星形多边形，让你判断一些点(其实就是羊)是否位于多边形内。</p> 

 
 # 输入格式 
<p>
第一行给出N,M表示羊圈羊圈点的个数及羊的数目<br>接下来N行，每行两个数x,y描述羊圈的顶点，顶点是按顺时针方向输入的.<br>接下来M行，每行两个数x,y描述羊的位置<br><br></p> 

 
 # 输出格式 
<p>
输出有m行，对于每只羊如果在羊圈内输出Good sheep,否则输出Bad sheep<br></p>
