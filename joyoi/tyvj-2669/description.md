 # 输入格式 
<p>
输入文件的第一行包含一个整数N，代表每队的人数。 <br>接下来的N行，每行一个整数，表示你的每个队员的能力值。 <br>接下来的N行，每行一个整数，依次表示对方团队的每个队员的能力值。 <br>最后一行包含一个整数，代表 L。 <br> <br> </p> 

 
 # 输出格式 
<p>
输出文件仅包含一个实数，表示你最大可能获得的分数。答案四舍五入保留<br>到小数点后六位 </p>
