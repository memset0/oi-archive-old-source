 # 输入格式 
<p>第一行输入两个数，分别表示行数和列数。</p>

<p>输入一个行数*列数的方阵，0表示面积（一个0表示1厘米），1表示无面积。</p> 

 
 # 输出格式 
<p>&nbsp;输出由方阵中0组成的三角形一个。</p>

<p>输出面积。</p>
