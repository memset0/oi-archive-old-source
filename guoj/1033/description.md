
# 题目描述

假设有 $ n $ 根柱子，现要按下述规则在这 $ n $ 根柱子中依次放入编号为 $ 1, 2, 3, 4, \cdots $ 的球。

1. 每次只能在某根柱子的最上面放球。
2. 在同一根柱子中，任何 $ 2 $ 个相邻球的编号之和为完全平方数。

试设计一个算法，计算出在 $ n $ 根柱子上最多能放多少个球。

# 输入格式

文件第 $ 1 $ 行有 $ 1 $ 个正整数 $ n $，表示柱子数。

# 输出格式

第一行是球数。接下来的 $ n $ 行，每行是一根柱子上的球的编号。

# 样例

#### 样例输入
```plain
4
```

#### 样例输出
```plain
11
1 8
2 7 9
3 6 10
4 5 11
```

# 数据范围与提示

$ 1 \leq n \leq 55 $
