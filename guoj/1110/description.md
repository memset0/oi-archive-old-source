
# 题目描述

白兔有一个长度为$n$的字符串。

白云有$m$个询问，每个询问会询问一段区间的本质不同回文子串个数。

# 输入格式

第一行两个整数$n, m$。

接下来一行一个长度为$n$的字符串。

接下来$m$行，每行两个整数$l, r$。

# 输出格式

为了避免输出占用太多运行时间，你只需要输出$\sum_{i=1}^m ans_i * i \bmod 10^9 + 7$即可。
