
# 题目描述

**这是一道提交答案题。**

外接圆在一个 $n\times m$ 的网格中。

其中有 $k$ 个格子是水泥路，剩下的部分是沼泽。外接圆只能走在水泥格子上，而且一旦他离开某一个水泥格子，那么那个格子上的水泥就会塌陷，变成沼泽。

如果外接圆某个时刻位于 $(x,y)$，那么腿短的外接圆下一时刻只能移动到与它四联通的格子，即 $(x-1,y),\ (x+1,y),\ (x,y-1),\ (x,y+1)$，且不能走出网格。

请为外接圆设计一个逃跑路线，使得他能在网格中待的时间最长。

# 输入格式

在附加文件中有 `escape1.in ∼ escape5.in` 共五个输入文件。

输入文件第一行两个正整数 $n$，$m$。

第二行两个正整数 $X$，$Y$，表示外接圆**第一个时刻**所在的位置。保证该位置为水泥路。

第三行一个正整数 $k$。

接下来 $k$ 行，第 $i$ 行两个正整数 $x_i$，$y_i$ 表示第 $x_i$ 行第 $y_i$ 列是水泥路。

保证不存在 $i,j$ 使得 $x_i=x_j$ 且 $y_i=y_j$。

# 输出格式

输出文件的第一行是路径的长度 $l$。

接下来 $l$ 行，第 $i$ 行两个正整数 $x_i$，$y_i$ 表示第 $i$ 个时刻外接圆在第 $x_i$ 行第 $y_i$ 列上。

# 样例

#### 样例 1 输入
```plain
2 2
4
1 1
1 2
2 1
2 2
```

#### 样例 1 输出
```plain
4
1 1
1 2
2 2
2 1
```

# 数据范围与提示

#### 评分方式

如果你的方案不合法，如**走出网格、走到沼泽上**等，你将会在这个测试点得到 $0$ 分。

否则，设你的方案中路径的长度为 $p$，标准答案中路径的长度为 $q$。

若 $0\le\frac pq<0.3$，你将会得到 $0$ 分。  
若 $0.3\le\frac pq<0.6$，你将会得到 $0.3\times$ 该测试点的满分。  
若 $0.6\le\frac pq<1$，你将会得到 $0.6\times$ 该测试点的满分。  
若 $\frac pq\ge 1$，你将会得到该测试点的满分。

|测试点编号|满分|
|:-:|:-:|
|$1$|$10$<!---->|
|$2$|$10$|
|$3$|$20$|
|$4$|$30$|
|$5$|$30$<!---->|

对于 $100\%$ 的数据，$1\le n,m,k\le 10^5$。
