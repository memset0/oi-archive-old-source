# 题目描述


<p>求所有可以只用1和00拼成的长度为N的二进制数的个数除以1 5746的余数。</p>
<p>比如当N=4的时候，有5个可能的二进制：0011，0000，1001，1100，1111。</p>
<h4>【输入格式】</h4>
<p>第一行一个正整数N</p>
<h4>【输出格式】</h4>
<p>输出所有可以只用1和00拼成的长度为N的二进制数的个数除以15746的余数。</p>
<h4>【输入样例】</h4>
<pre>4</pre>
<h4>【输出样例】</h4>
<pre>5</pre>
<h4>【数据范围】</h4>
<p>在100％的数据中，1≤N&lt;1000000</p>
