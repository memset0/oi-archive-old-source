# 题目描述


<h3>
【题目描述】
</h3>
<p>
给你一个无向带权连通图，每条边是黑色或白色。让你求一棵最小权的恰好有need条白色边的生成树。<br/>
题目保证有解。
</p>
<h3>
【输入格式】
</h3>
<p>
第一行V,E,need分别表示点数，边数和需要的白色边数。<br/>
接下来E行<br/>
每行s,t,c,col表示这边的端点(点从0开始标号)，边权，颜色(0白色1黑色)。
</p>
<h3>
【输出格式】
</h3>
<p>
一行表示所求生成树的边权和。
</p>
<h3>
【数据规模和约定】
</h3>
<p>
0:V&lt;=10<br/>
1,2,3:V&lt;=15<br/>
0,..,19:V&lt;=50000,E&lt;=100000<br/>
所有数据边权为[1,100]中的正整数。
</p>
<h3>
【样例输入】
</h3>
<pre>2 2 1
0 1 1 1
0 1 2 0
</pre>
<h3>
【样例输出】
</h3>
<pre>2
</pre>

【题目来源】
</h3>
<p>

</p>
