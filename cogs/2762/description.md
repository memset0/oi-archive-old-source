# 题目描述


<h3>
【题目描述】
</h3>
<p>
输入一个正整数$N$，求$N$以内所有的素数之和对于unsigned long long自然溢出后的结果。
</p>
<h3>
【输入格式】
</h3>
<p>
第一行一个整数$N$
</p>
<h3>
【输出格式】
</h3>
<p>
一行一个素数
</p>
<h3>
【样例输入】
</h3>
<pre>10</pre>
<h3>
【样例输出】
</h3>
<pre>17
</pre>
<h3>
【提示】
</h3>
<p>
数据范围：$10\leq N\leq 100000000$
</p>
