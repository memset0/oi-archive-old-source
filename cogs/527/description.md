# 题目描述


<h3>
【题目描述】
</h3>
<p>
给出圆周上的若干个点，已知点与点之间的弧长，其值均为正整数，并依圆周顺序排列。 <br/>
请找出这些点中有没有可以围成矩形的，并希望在最短时间内找出所有不重复矩形。
</p>
<h3>
【输入格式】
</h3>
<p>
第一行为正整数N，表示点的个数，接下来N行分别为这N个点所分割的各个圆弧长度
</p>
<h3>
【输出格式】
</h3>
<p>
所构成不重复矩形的个数
</p>
<h3>
【样例输入】
</h3>
<pre>8
1
2
2
3
1
1
3
3
</pre>
<h3>
【样例输出】
</h3>
<pre>3
</pre>
<h3>
【提示】
</h3>
<p>
N≤20 <br/>
