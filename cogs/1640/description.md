# 题目描述


<h3>
【题目描述】
</h3>
<p>
输入一个n×n的黑白图像（1表示黑色，0表示白色），任务是统计其中八连块的个数。如果两个黑格子有公共边或者公共顶点，就说它们属于同一个八连块。如下图所示的图形有3个八连块。
</p>
<p>
<img src="/upload/image/20140523/20140523204138_97566.jpg" alt=""/> 
</p>
<h3>
【输入格式】
</h3>
<p>
第1行输入一个正整数n（n≤700），此后输入n行，每行是由n个0或1组成的字符串。
</p>
<h3>
【输出格式】
</h3>
<p>
在此键入。
</p>
<h3>
【样例输入】
</h3>
<pre>6
100100
001010
000000
110000
111000
010100
</pre>
<h3>
【样例输出】
</h3>
<pre>3
</pre>
