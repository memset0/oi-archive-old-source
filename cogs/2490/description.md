# 题目描述


<h3>
【题目描述】
</h3>
<p>
</p><p>
一本书的页码是从 1-n 编号的连续整数： 1, 2, 3, ... , n。请你求出全部页码中
</p>
<p>
所有单个数字的和，例如第 123 页，它的和就是 1+2+3=6。
</p>
<p>
<br/>
</p>
<h3>
【输入格式】
</h3>
<p>
</p><p>
一行为 n(1 &lt;= n &lt;= 10^9)。
</p>
<p>
<br/>
</p>
<h3>
【输出格式】
</h3>
<p>
</p><p>
一行，代表所有单个数字的和。
</p>
<p>
<br/>
</p>
<h3>
【样例输入】
</h3>
<pre><p>
3456789
</p>
</pre>
<h3>
【样例输出】
</h3>
<pre><p>
96342015
</p>
</pre>
