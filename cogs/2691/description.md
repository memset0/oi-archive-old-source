# 题目描述


<h3>
【题目描述】
</h3>
<p>
考虑两个自然数A和B.定义S是$A ^ B$的所有自然因数的总和。确定S模9901的值。
</p>
<h3>
【输入格式】
</h3>
<p>
唯一的行包含由空格分隔的两个自然数A和B（0 &lt;= A，B &lt;= 50000000）。
</p>
<h3>
【输出格式】
</h3>
<p>
输出一行，即S模9901。
</p>
<h3>
【样例输入】
</h3>
<pre>2 3</pre>
<h3>
【样例输出】
</h3>
<pre>15</pre>
<p>
样例解释：2 ^ 3 = 8。 8的自然因数是：1,2,4,8。他们的和是15。 15模9901是15（应该输出）。
</p>
