# 题目描述


<h2>
	【问题描述】
</h2>
<p>
	两伙外星人策划在未来的XXXX年侵略地球,侵略前自然要交换信息咯,现在,作为全球保卫队队长,你截获了外星人用来交换信息的一段仅由’F’,’B’,’I’,’O’,组成的序列,为了保卫地球和平,为了使家园不受破坏,你要机智地破解密码,勇敢地迎击外星人!记住,你不是一个人在战斗!你不是一个人!你的背后是千千万万的地球人！
</p>
<h2>
	【输入文件】
</h2>
<p>
	    一组仅由’F’,’B’,’I’,’O’,组成的序列（’F’,’B’,’I’,’O’这四个字母中的某一个或某几个不一定会出现，且保证序列长度&lt;=2000) <br/>
规定这个序列所要传达的信息就是这组序列有多少个’FBI’(子序列) 
</p>
<h2>
	【输出文件】
</h2>
<p>
	一个数,表示这组序列有多少个’FBI’的子序列(保证答案&lt;=2^31，且FBI必须是正序，即IBF或者BIF或者FIB或者BFI或者IFB都不能算是一个FBI)
</p>
<h2>
	【样例输入】
</h2>
<p>
	FBIIBFOI
</p>
<h2>
	【样例输出】
</h2>
<p>
	4
</p>
<h2>
	【样例说明】
</h2>
<p>
	第一个FBI是由序列的第一个字母+序列的第二个字母+序列的第三个字母组成 <br/>
第二个FBI是由序列的第一个字母+序列的第二个字母+第四个字母组成 <br/>
第三个FBI是由序列的第一个字母+序列的第二个字母+第八个字母组成 <br/>
第四个FBI是由序列的第一个字母+序列的第五个字母+第八个字母组成
</p>
