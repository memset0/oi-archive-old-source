# 题目描述


<h3>
【题目描述】
</h3>
<p>
输入x和n，计算并输出f(x,n)的值。
</p>

<h3>
【输入格式】
</h3>
<p>
输入只有一行，包含两个数x,n，其中x为0~100的实数，n为0~20的整数。
</p>
<h3>
【输出格式】
</h3>
<p>
输出f(x,n)的值，结果保留两位小数。
</p>
<h3>
【样例输入】
</h3>
<pre>4.2 10</pre>
<h3>
【样例输出】
</h3>
<pre>3.68</pre>
