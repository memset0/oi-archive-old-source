# 题目描述


<h3>
【题目描述】
</h3>
<p>
请从[1,n]中选三个数，可以重复，组成一个等比数列，求这样的等比数列个数。两个等比数列不同当且仅当其中的任意一位不同。
</p>
<h3>
【输入格式】
</h3>
<p>
一行一个整数n。
</p>
<h3>
【输出格式】
</h3>
<p>
输出一个整数，表示答案，数据保证答案不超过long long范围
</p>
<h3>
【样例输入】
</h3>
<pre>10
</pre>
<h3>
【样例输出】
</h3>
<pre>18
</pre>
<h3>
【提示】
</h3>
<p>
数据范围n&lt;=1e14，出题人太懒，不设部分分。
</p>

【来源】
