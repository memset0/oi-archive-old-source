# 题目描述


<h1>
	Problem C. 短
</h1>
给出无向图G = (V, E),求点V1 和点VN 之间的最短路。<br/>
<h3>
	Input Specification
</h3>
第1行,2个整数|V |和|E|。<br/>
第2行到第|E| + 1行,3个整数xi ,yi ,zi ,表示点Vxi 和点Vyi 之间存在权值为wi 的边。<br/>
<h3>
	Output Specification
</h3>
第1行,1个整数,表示最短路的权值。<br/>
<h3>
	Input and Output Sample
</h3>
duan.in<br/>
3<br/>
1<br/>
2<br/>
3<br/>
3<br/>
2 1<br/>
3 1<br/>
1 1<br/>
duan.out<br/>
1<br/>
<h3>
	Data Restriction
</h3>
对于50%的数据,N ≤ 1, 000。<br/>
对于100%的数据,N ≤ 100, 000, M ≤ 1, 000, 000, 1 ≤ Wi ≤ 2。<br/>
