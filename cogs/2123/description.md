# 题目描述


<h3>
【题目描述】
</h3>
<p>
给定长度为n（n&lt;=300000）的循环同构的字符串，定义最小表示为该字符串的字典序最小的同构表示，请输出这个表示。
</p>
<h3>
【输入格式】
</h3>
<p>
第一行是串的长度，第二行是字符串。
</p>
<h3>
【输出格式】
</h3>
<p>
串的最小表示。
</p>
<h3>
【样例输入】
</h3>
<pre>10
helloworld
</pre>
<h3>
【样例输出】
</h3>
<pre>dhelloworl
</pre>
