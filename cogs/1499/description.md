# 题目描述


<h3>
【题目描述】
</h3>
<p>
解方程：
</p>
<p>
$pe^{-x} + q\sin(x) + r\cos(x) + s\tan(x) + tx^2 + u = 0$
</p>
<p>
其中0&lt;=x&lt;=1.
</p>
<h3>
【输入格式】
</h3>
<p>
输入包含多组数据。
</p>
<p>
每组数据有1行，包含6个整数：$p,q,r,s,t,u$
</p>
<p>
其中$0&lt;=p,r&lt;=20,-20&lt;=q,s,t&lt;=0$
</p>
<h3>
【输出格式】
</h3>
<p>
对每组数据，输出一行，即方程的解，精确到四位小数，若无解则输出&#34;No solution&#34;。
</p>
<h3>
【样例输入】
</h3>
<pre>0 0 0 0 -2 1
1 0 0 0 -1 2
1 -1 1 -1 -1 1
</pre>
<h3>
【样例输出】
</h3>
<pre>0.7071
No solution
0.7554
</pre>
<h3>
【提示】
</h3>
<p>
数据组数&lt;=2100
</p>
<p>
使用“保留4位小数”的命令，假设你的计算机能正确进行舍入。
</p>
