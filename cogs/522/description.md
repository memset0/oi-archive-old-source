<p><br/>
最佳地点<br/>
<br/>
贝茜总是希望优化她的人生，她很想访问约翰农场所拥有的P(1 &lt;= F &lt;= P)个草地中的F(1 &lt;= P &lt;= 500; 1 &lt;= F_i &lt;= P)个。<br/>
贝茜知道,她只能在C (1 &lt;= C &lt;= 8,000)条双向的路径上旅行，在路径P_i旅行，需要时间T_i(1 &lt;= T_i &lt;= 892),路径的两个端点是a_i and b_i (1 &lt;= a_i &lt;=P; 1 &lt;= b_i &lt;= P).<br/>
贝茜想找一个最好的草地睡觉，并且到达这个草地能让这次旅行的平均时间最短。<br/>
例如，下面是一个农场的草地分布图，带*号的草地是贝茜特别喜欢的，括号中的数是该路径的旅行时间。<br/>
