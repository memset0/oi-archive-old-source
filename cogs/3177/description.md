# 题目描述


<h3>
【题目描述】
</h3>
<p>
小$ D$ 想知道， {$1$, $2$, $3$, ..., $n$} 的所有排列中，有多少个排列的逆序对恰好为$ k$ 个。
</p>
<p>
由于小 $D$可能很喜欢 $10^9$ + $7$，所以答案对其取模。
</p>
<h3>
【输入格式】
</h3>
<p>
第一行，正整数 $n$，自然数 $k$。
</p>
<h3>
【输出格式】
</h3>
<p>
输出一行，一个整数，代表取模后的答案。
</p>
<h3>
【样例输入】
</h3>
<pre>3 2
</pre>
<h3>
【样例输出】
</h3>
<pre>2
</pre>
<h3>
【提示】
</h3>
<p>
保证$ n $≤ $10^5$，$ k $≤ $min$($10^5$,$C(n,2)$)。
</p>
