# 题目描述


<p>【问题描述】<br/>
    给定一个N*M方格的迷宫，迷宫里有T处障碍，障碍处不可通过。给定起点坐标和终点坐标，问每个方格最多经过1次，有多少种从起点坐标到终点坐标的方案。在迷宫中移动有上下左右四种方式。保证起点上没有障碍。<br/>
【输入文件】<br/>
第一行N、M和T，N为行，M为列，T为障碍总数。<br/>
第二行起点坐标SX,SY，终点坐标FX,FY。<br/>
接下来T行，每行为障碍的坐标。<br/>
【输出文件】<br/>
给定起点坐标和终点坐标，问每个方格最多经过1次，从起点坐标到终点坐标的方案总数。<br/>
 </p>
<p>【样例输入:maze.in】<br/>
2 2 1<br/>
1 1 2 2<br/>
1 2<br/>
【样例输出:maze.out】<br/>
1<br/>
【数据规模】<br/>
1&lt;=N,M&lt;=5。</p>
