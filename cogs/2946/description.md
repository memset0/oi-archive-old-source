# 题目描述


<h3>
【题目描述】
</h3>
<p>
对于 C 语言的循环语句 for(i=A;i!=B;i+=C)，需要循环多少次才会在 $k$ 位的储存系统中结束，输出循环的次数，若无法结束，则输出“FOREVER”。$1\le k\le 32$，$0\le A,B,C\lt 2^k$。
</p>
<h3>
【输入格式】
</h3>
<p>
输入包含多组数据，每组数据由一行四个整数 $A,B,C,k$ 组成。输入以四个零结尾。
</p>
<h3>
【输出格式】
</h3>
<p>
每组数据输出一行一个整数，表示循环的次数，无法结束则输出“FOREVER”。
</p>
<h3>
【样例输入】
</h3>
<pre>3 3 2 16
3 7 2 16
7 3 2 16
3 4 2 16
0 0 0 0
</pre>
<h3>
【样例输出】
</h3>
<pre>0
2
32766
FOREVER
</pre>
