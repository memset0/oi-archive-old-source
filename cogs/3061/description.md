# 题目描述


<h3>
【题目描述】
</h3>
<p>
用 n 根火柴组成数字，要求输出最大非负数与最小非负数。
</p>
<p>
火柴数字如下：
</p>
<p>
<img src="https://cdn.luogu.org/upload/pic/49.png" alt=""/> 
</p>
<h3>
【输入格式】
</h3>
<p>
一个数 n 。
</p>
<h3>
【输出格式】
</h3>
<p>
最大非负数与最小非负数，中间用空格隔开。
</p>
<h3>
【样例输入】
</h3>
<pre>7</pre>
<h3>
【样例输出】
</h3>
<pre>711 8</pre>
<h3>
【提示】
</h3>
<p>
2&lt;=n&lt;=100000。
</p>
