# 题目描述


<h3>
【题目描述】
</h3>
<p>
寻找一个由n个整数组成的数列，其中任意连续p个整数之和为正，任意连续q个整数之和为负。若不存在这样的整数数列，则输出no，否则输出其中一个数列。
</p>
<h3>
【输入格式】
</h3>
<p>
输入文件只有一行，三个整数n,p,q(0&lt;n&lt;=100，q&lt;n，p&lt;n，p&lt;&gt;q)
</p>
<h3>
【输出格式】
</h3>
<p>
一行，n个整数，中间用一个空格隔开，表示符合题目要求的序列(如果这样的序列不存在，则输出no)
</p>
<h3>
【样例输入】
</h3>
<pre>6 5 3</pre>
<h3>
【样例输出】
</h3>
<pre>-3 5 -3 -3 5 -3</pre>
<br/>
