# 题目描述


<h3>
【题目描述】
</h3>
<p>
给定整数 N(1≤N≤10^6)，试把阶乘$ N!$ 分解质因数，按照算术基本定理的形式输出分解结果中的 $p_i$ 和 $c_i$ 即可。
</p>
<h3>
【输入格式】
</h3>
<p>
一个整数N。
</p>
<h3>
【输出格式】
</h3>
<p>
N! 分解质因数后的结果，共若干行，每行一对pi, ci，表示含有$pi^{ci}$项。按照pi从小到大的顺序输出。
</p>
<h3>
【样例输入】
</h3>
<pre>5</pre>
<h3>
【样例输出】
</h3>
<pre>2 3
3 1
5 1
</pre>
<h3>
【提示】
</h3>
<p>
5! = 120 = 2^3 * 3 * 5
</p>
<h3>
【来源】
</h3>
<p>
《算法竞赛进阶指南》
</p>
