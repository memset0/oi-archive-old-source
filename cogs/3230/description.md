# 题目描述


<h3>
【题目描述】
</h3>
<div class="content">
<p>
已知平面内 N 个点的坐标，求欧几里德距离下的第 K 远点对。
</p>
<div>
</div>
</div>
<h3>
【输入格式】
</h3>
<div class="content">
<div>
输入文件第一行为用空格隔开的两个整数 N， K。接下来 N 行，每行两个整数 X,Y，表示一个点
</div>
<div>
的坐标。1 &lt; =  N &lt; =  100000， 1 &lt; =  K &lt; =  100， K &lt; =  N*(N−1)/2 ， 0 &lt; =  X, Y &lt; 2^31。
</div>
<div>
</div>
</div>
<h3>
【输出格式】
</h3>
<div class="content">
<p>
输出文件第一行为一个整数，表示第 K 远点对的距离的平方（一定是个整数）。
</p>
<div>
</div>
</div>
<h3>
【样例输入】
</h3>
<pre>10 5
0 0
0 1
1 0
1 1
2 0
2 1
1 2
0 2
3 0
3 1</pre>
<h3>
【样例输出】
</h3>
<pre>9</pre>
