# 题目描述


<h3>
【题目描述】
</h3>
<p>
求Pell方程x^2-n*y^2=1的最小解
</p>
<h3>
【输入格式】
</h3>
<p>
一个正整数N（N&lt;=30）
</p>
<h3>
【输出格式】
</h3>
<p>
两个正整数x0,y0代表所给Pell方程的最小解
</p>
<h3>
【样例输入】
</h3>
<pre>3</pre>
<h3>
【样例输出】
</h3>
<pre>2 1</pre>
