# 题目描述


<h3>
【题目描述】
</h3>
<p>
题目描述:求C(N,M) mod P,P可能为合数,1&lt;=M&lt;=N&lt;=10^7,P&lt;=10^7
</p>
<h3>
【输入格式】
</h3>
<p>
共一行,三个数依次为:N,M,P.
</p>
<h3>
【输出格式】
</h3>
<p>
共一行,一个数即C(N,M) mod P.
</p>
<h3>
【样例输入】
</h3>
<pre>5 4 3</pre>
<h3>
【样例输出】
</h3>
<pre>2</pre>
