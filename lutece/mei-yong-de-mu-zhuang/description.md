
# Content

某农场想要利用已有的n个木桩建篱笆，由于农场主非常信玄学，所以请了一位风水大师来商量要如何修建，风水大师对于这种情况有着独特的计算方法：如果要使得一个木桩在所有木桩的凸壳上，那么至少需要移除多少个木桩呢？

# Standard Input

第一行包含一个整数n，表示点的个数。
接下来n行每行包含两个整数x，y，表示点的坐标。
1 <= n <= 3000，-10^6 <= x, y <= 10^6。

# Standard Output

输出n行，按输入顺序输出每个点对应的答案。
