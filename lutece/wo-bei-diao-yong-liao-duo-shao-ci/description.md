
# Content

编写一个函数，该函数能够判断传递过来的数是否是素数，并返回自身是被第几次调用。要求不用全局变量。

# Standard Input

第一行是整数$n$，表示下一行数据的个数，$1 \leq n \leq 1000$，每个数据的位数不超过$6$位。

# Standard Output

输出两个整数，用空格隔开。第一个数表示判断出的素数个数，第二个数表示函数被调用的次数。
