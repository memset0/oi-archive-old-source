
# Content

读入一个整数$n$和一个字符$ch$，输出边长为$n$，且全由字符$ch$组成的菱形。具体见样例。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 100$），表示测试数据的组数，随后有$T$组测试数据。每一组测试数据只占一行，由整数$n$和字符$ch$组成，两者之间有一个空格。$0<n\le 100$。

# Standard Output

对应每组输入，输出一个边长为$n$，由字符$ch$组成的菱形。
