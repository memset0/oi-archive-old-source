
# Content

写一个函数，生成对顶三角形，本题目要求不用全局变量。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 100$），表示测试数据的组数。每一组测试数据只有一行，分别是三角形的腰长$n$和构成三角形的字符$ch$，两者之间有一个空格。该行没有其它多余的符号。$0<n\le 100$。

# Standard Output

对应每组输入，输出一个对顶三角形。每组数据输出后应有一个空行。
