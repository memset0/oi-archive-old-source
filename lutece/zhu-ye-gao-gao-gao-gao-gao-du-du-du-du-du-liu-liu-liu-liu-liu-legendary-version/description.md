
# Content

有一条长度为$N$的线段,初始每个位置颜色都是$0$.

有$K$次操作,第$i$次操作可以任选一段线段上的非空区间并将这个选择的区间颜色改为$i$.

求$K$次操作后,一共有多少种可能的线段?

输出取模$163577857$后的答案.

# Standard Input

输入只有一行有一个整数$T$,表示数据组数.

接下来$T$行，每行两个整数$N$和$K$.

数据保证:
* $1 \leq T \leq 100$
* $1 \leq N \leq 10^6$
* $1 \leq K \leq N$
* $1 \leq \sum N \leq 5 \cdot 10^6$

# Standard Output

对每组数据，输出答案.
