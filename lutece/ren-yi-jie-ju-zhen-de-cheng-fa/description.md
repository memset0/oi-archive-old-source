
# Content

求两个任意阶矩阵的乘法。

# Standard Input

第一行是测试数据的组数$T$。对于每一组数据，第一行有三个整数$a$、$b$、$c$，表示输入的两个矩阵阶数分别为$a$`*`$b$和$b$`*`$c$。每个元素数据的范围为[$-1000,1000$]，矩阵的维数不会超过$10$。

# Standard Output

每组数据对应有一个矩阵输出，每个数后输出一个空格。

每组数据之后输出一个空行。
