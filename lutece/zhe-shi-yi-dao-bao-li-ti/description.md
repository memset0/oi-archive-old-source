
# Content

给你$n$种数字，请你找到一个由这些数字组成的一个最大的能被$3$整除的$k$位数，如果不存在输出$-1$

# Standard Input

第一行两个数$n,k(1 \leq n \leq 10, 1\leq k \leq 1e5)$    
第二行有$n$个互不相同的数字，范围$[0,9]$

# Standard Output

输出一行，最大的满足条件的数，不存在输出$-1$
