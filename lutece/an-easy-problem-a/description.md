
# Content

N个数排成一列，Q个询问，每次询问一段区间内的数的极差是多少。

# Standard Input

第一行两个整数N(1≤N≤50000),Q(1≤Q≤200000)。接下来一行N个整数a1 a2 a3 ....an,(1≤ai≤1000000000)。接下来Q行，每行两个整数L,R(1≤L≤R≤N)。

# Standard Output

对于每个询问输出一行，一个整数表示区间内的极差。
