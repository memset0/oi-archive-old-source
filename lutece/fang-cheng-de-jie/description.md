
# Content

给定一个四元二次方程：

$AX_1^2+BX_2^2+CX_3^2+DX_4^2=0$

试求$-1000 \le X_1,X_2,X_3,X_4 \le 1000$非零整数解的个数。

# Standard Input

第一行四个整数$A,B,C,D$

$-10000 \le A,B,C,D \le 10000$

# Standard Output

第一行一个整数$x$，表示方程整数解的个数。
