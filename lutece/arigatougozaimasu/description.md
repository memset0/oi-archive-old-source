
# Content

在数字王国，一个数可以选择以下四种魔法之一变成另一个数：

$1$. $x$变为$x+2$ 消耗$1$点魔法值

$2$. $x$变为$x-3$  消耗$2$点魔法值

$3$. $x$变为$x*4$ 消耗$3$点魔法值

$4$. 如果$x$是$5$的倍数，$x$变为$x/5$ 消耗$4$点魔法值.

现给你$N,M$,请问你要把$N$通过魔法变成$M$，最少需要消耗多少魔法值？

# Standard Input

一行两个整数$N,M$.

$(1<=N,M<=100000)$

# Standard Output

把$N$变成$M$最少需要消耗的魔法值。
