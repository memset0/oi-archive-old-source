
# Content

一行字符串可能含有多个正整数, 请找出这些整数, 并输出它们的和.

# Standard Input

有多组测试数据. 输入的第一行是整数T(1<=T<=100), 表示随后测试数据的组数. 每组测试数据占一行, 由一个可能含有空格的字符串构成, 该字符串可能含有多个数字子串, 每个数字子串的数字个数不超过16, 数字子串的个数不超过10, 字符串的长度不超过1000.

# Standard Output

对应每组测试数据, 输出该字符串所包含整数的和. 如果该字符串没有包含数字, 请输出0.
