
# Content

读入$4$个整数$a$、$b$、$c$、$d$和一个运算符（`+`、`-`、`*`、`/`中的一个），进行分数$\frac{a}{b}$和$\frac{c}{d}$的对应运算，输出最简结果。

# Standard Input

有多组测试数据。输入的第一行是整数$T$($1\leq T\leq 200$)，表示随后测试数据的组数。每组测试数据占一行，由一个四个正整数$a$,$b$,$c$,$d$和一个运算符组成，相邻数据间有一个空格。

# Standard Output

对应每组测试数据，输出分数运算的最简结果，占一行。具体可参照样例。
