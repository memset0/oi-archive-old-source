
# Content

这一次要求用动态分配内存的方式实现任意阶整数矩阵的乘法。矩阵乘法也要求用函数实现。

# Standard Input

第一行是测试数据的组数$T$。对于每一组数据，第一行有三个用空格隔开的整数$a$、$b$、$c$，表示输入的两个矩阵阶数分别为$a\times b$和$b\times c$，紧接着是两个矩阵的元素。每个元素数据的范围为$[-1000,1000]$。

# Standard Output

每组数据对应有一个矩阵输出，每个数后输出一个空格。每组结果之后输出一个空行。
