
# Content

Calculate $N!$ mod $(10^{11}+3)$.

# Standard Input

The first line contains an integer $T$, denoting the number of the test cases.

Each test case contains one number $N$.

$1\leq T\leq 5, 1\leq N\leq 10^{11}+3$

# Standard Output

$N!$ mod $(10^{11}+3)$.
