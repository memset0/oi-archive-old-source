
# Content

魔法师`张兑田`想要制造分别$x,y,z$个黄球，绿球，蓝球。  
- 制造一个黄球需要两个黄色水晶。  
- 制造一个绿球需要一个黄色水晶和一个蓝色水晶。  
- 制造一个蓝球需要三个蓝色水晶。  

现在魔法师已经有了$A$个黄色水晶和$B$个蓝色水晶，请问还需要收集多少个水晶(包括蓝水晶和黄水晶)。  

聪明的你，可以帮她解决这个问题吗?

# Standard Input

输入包括一行五个数$A,B,x,y,z$，具体含义如上文。  
$0\leq A,B,x,y,z\leq 100$

# Standard Output

输出一个数表示还需要收集多少个水晶。
