这时她突然想起，曜酱似乎跟她说过，矩阵的秩与矩阵是否可逆、线性方程组是否有解有着密切的关系……

（矩阵秩的定义请自行搜索）

# Standard Input

输入包含多组数据，以EOF结尾。   
每组数据第一行包含两个正整数n，m，代表给定矩阵的行数和列数。    
接下来n行，每行m个浮点数，代表给定的矩阵的元素。

# Standard Output

对于每个给定的矩阵，单独输出一行，包含一个整数，代表该矩阵的秩。
