# Standard Input

The first line of the input contains an integer $N(1\leq N\leq100000)$ - the number of points. The following N lines contain pairs of positive integer $X$ and $Y$ $(1\leq X,\ Y\leq100000)$ - the coordinates of the points.

# Standard Output

Output one floating-point number - the minimal area of the triangle. The answer is considered to be correct if its absolute or relative error does not exceed $10^{-6}$.
