
# Content

平面上有N个红点和N个蓝点，求红点到蓝点的最近距离

# Standard Input

第一行为一个整数N  $1\le N \le 100000$
接下来第N行每行两个整数xi,yi，表示第i个红点的坐标  
接下来第N行每行两个整数xi,yi，表示第i个蓝点的坐标($1\le x,y \le 100000$)

# Standard Output

红点到蓝点的最短距离
