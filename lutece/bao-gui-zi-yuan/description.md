
# Content

有一个城市需要建造，给你$N$个矿场的坐标$X$,$Y$，问把这么多矿坑全都包进城市的话，城市所需最小面积是多少（注意，城市为平行于坐标轴的正方形）

# Standard Input

第一行为$N$，表示矿场数目

下面为$N$行  每行两个数字$x_i$，$y_i$，表示第$i$行表示第$i$个矿场的坐标

$2≤N≤1000$

$0≤x_i,y_i≤1000 000 000$

# Standard Output

城市所需最小面积
