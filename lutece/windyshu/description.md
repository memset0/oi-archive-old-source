
# Content

windy定义了一种windy数。

不含前导零且相邻两个数字之差至少为$2$的正整数被称为windy数。

windy想知道，在$A$和$B$之间，包括$A$和$B$，总共有多少个windy数？

# Standard Input

包含两个整数，$A$ $B$。

满足 $1 \leq A \leq B \leq 2000000000$ .
