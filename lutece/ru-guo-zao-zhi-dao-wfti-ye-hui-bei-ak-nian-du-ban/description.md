定义数列A由以下规则生成：\
A[i]=(A[i-1]+A[i-2]+A[i-3])%p\
A[1]=1\
A[2]=1\
A[3]=2\
给出A的某个连续段A[i-1],A[i],A[i+1]，求最小的满足条件的i

# Standard Input

第一行有三个整数A[i-1], A[i], A[i+1]

# Standard Output

输出一行，包含一个整数，表示最小的满足条件的i
