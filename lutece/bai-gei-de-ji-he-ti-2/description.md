
# Content

给定一个平面，存在一些点，询问点对间最远的一对的距离是多少

# Standard Input

第一行一个数字n，代表点的数量

接下来n行，每行两个数字，分别表示每个点的x和y坐标

# Standard Output

一个数字x，最远点距离的平方
