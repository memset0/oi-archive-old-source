
# Content

把给定的英文文章各段中，奇数位的英文字母转换为大写，偶数位的英文字母转换为小写。

注意，奇偶计数按整篇文章计算，回车不计入内。

# Standard Input

输入英文文章有多段，其中不仅包含了英文字母，可能还包含了其它字符。

# Standard Output

输出转换后的英文文章。
