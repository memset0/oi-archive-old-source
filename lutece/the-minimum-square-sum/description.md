
# Content

Given a prime $p (p<10^8)$,you are to find min{$x^2+y^2$},where $x$ and $y$ belongs to positive integer, so that $(x^2+y^2) \equiv 0$ (mod $p$).

# Standard Input

Every line is a $p$. No more than $10001$ test cases.

# Standard Output

The minimum square sum as described above.
