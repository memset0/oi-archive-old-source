
# Content

读入一个自然数，将$n$分解为质因子连乘的形式输出。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 10000$），表示测试数据的组数。每一组测试数据只有一行，包含待分解的自然数$n$。该行没有其它多余的符号。$1<n<2^{31}$。

# Standard Output

对应每组输入，输出一行分解结果，具体样式参看样例。该行不能有其它多余的符号。
