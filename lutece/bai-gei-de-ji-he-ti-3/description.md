
# Content

给出N个点，让你画一个最小的包含所有点的圆。

# Standard Input

多组读入，直到读入结束为止。

对于每一组数据，第一行一个n，表示点的数量。

# Standard Output

第一行输出圆的半径，第二行圆心的坐标，严格保留3位小数。
