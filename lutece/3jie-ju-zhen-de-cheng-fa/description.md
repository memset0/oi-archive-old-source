
# Content

实现两个$3$*$3$矩阵的乘法。

# Standard Input

第一行是一个正整数$n$表示测试数据的组数。接下来有$2n$个三阶矩阵。整数的范围为[$-1000,1000$]。

# Standard Output

每组数据对应有一个矩阵输出，每个数后输出一个空格。

每组数据之后输出一个空行。
