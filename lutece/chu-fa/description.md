
# Content

输入正整数$n$，按从小到大的顺序输出所有形如`abcde/fghij=n`的表达式，其中`a`~`j`恰好为数字$0\sim 9$的一个排列，$2\le n\le 79$。

# Standard Input

输入数据只包括一个数字$n$。

# Standard Output

输入对应的输出，若输出有多个，则一个占一行，且依据分母从小到大的顺序。如果没有对应的结果，则输出一行`NO`。
