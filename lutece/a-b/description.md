
# Content

求$a$的$b$次方后四位。

# Standard Input

输入的第一行是$T$（不超过$1000$）。$T$表示测试部分的个数，每一部分都要求单独计算并按照要求输出结果。

接下来是每个测试部分。第一行给出$a$, $b$，$0 < a,b\leq 1000000000$。

# Standard Output

对于每个测试部分，请输出每个问题的正确答案的最后四位，如果不足四位请前面补$0$。
