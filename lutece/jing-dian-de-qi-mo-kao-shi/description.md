
# Content

京电要期末考试啦！！！！！  
京电和C电不一样，京电的组合数学考试是电脑上完成的。  
老师给你一个01串，你需要所有找出所有的非空子序列使得子序列的前一半都是0，后一半都是1(选择的子序列的长度必须是偶数)。  
女神给了你一个眼神，作为老实人，你会帮你的女神做出这题的对吧。  
对了，女神希望把最后的结果mod 1e9+7 这样才不用花太多时间去记，有时间和其他男生看电影吃饭呢。

# Standard Input

第一行一个01字符串,其长度不超过200,000

# Standard Output

子序列的数量 mod 1e9+7 呢
