
# Content

电子科技大学一学期一度的期末考试终于结束了。但是在考完模电（魔鬼电路）之后，集训队内的LoveZx十分忐忑。曾经Zx与LoveZx约定，如果LoveZx的模电成绩能够超过她，那么LoveZx会拥有一个很幸福的寒假。为了自己的幸福，LoveZx通过各方努力预先得知了自己与Zx的成绩，又由于自己与教务处的关系良好，LoveZx可以通过贿赂来修改自己的成绩。每次贿赂之后，LoveZx可以选择自己成绩的任何一位（十进制）往前移动到任意位置。比如`123`，LoveZx可以通过一次贿赂将它变成`213`或`312`或`132`。当然啦，贿赂是需要资金的。现在LoveZx求助于你，请你让他以最少的贿赂次数，获得一个幸福的寒假。

# Standard Input

输入包含多组数据。第一行是一个整数T表示数据组数（$0< T\leq 50$）。

接下来每组数据包括一行，即两个整数$A$,$B$（不含前导零），分别是Zx与LoveZx的成绩（$0\leq A,B\leq 10^{100000}$）。

# Standard Output

对每组测试数据输出一行。

每行仅输出一个整数，表示最少的贿赂次数，如果无论如何LoveZx都无法拥有一个幸福的寒假，输出`Poor LoveZx`（不包含引号）。
