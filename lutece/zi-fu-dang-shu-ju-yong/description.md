
# Content

读入三个字符，把它们ASCII码当作整数，求平均值并输出。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 1000$），表示测试数据的组数。每一组测试数据只有一行，分别为字符$a$、$b$和$c$，三字符间没有空格。该行没有其它多余的符号。

# Standard Output

对应每组输入，输出一行三个字符ASCII码的平均值，保留两位小数。该行不能有其它多余的符号。
