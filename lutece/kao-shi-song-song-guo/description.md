数$N$是五角数，是指$N$个点可以被排在一个（填满的）五边形里。

前$4$个五角数分别是：$1,5,12$和$22$.

# Standard Input

多组测试数据。

每组数据输入只有一个整数$M$（$0 \leq M < 10^7$）

# Standard Output

每组测试数据输出一个数，表示第$M+1$个五角数$N$是多少。
