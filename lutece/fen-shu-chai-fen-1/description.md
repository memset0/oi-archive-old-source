
# Content

输入正整数$k$，找到所有的正整数$x \leq y$，使得$1/k = 1/x + 1/y$，其中$2 \leq k \leq 10000$

# Standard Input

输入共一行，代表正整数$k$

# Standard Output

输出共$N$行

每行是一组满足$1/k = 1/x + 1/y$的等式

请按照$x$递增的顺序输出（等式中间没有空格）
