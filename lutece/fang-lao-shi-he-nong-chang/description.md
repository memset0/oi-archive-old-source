
# Content

方老师重新开了一家农场，农场一共有$N$个节点和$M$条双向边，但是有一个很大的问题就是有比如一个农场$A$到达农场$B$只有一条路径，问至少添加多少条边使得任意两个农场之间的路径多于一条。

# Standard Input

* 多组数据，`EOF`结束。
* 第$1$行：$N$和$M$。
* 第$2$到第$M+1$行：每一行$2$个数$U_i$和$V_i$，表示$U_i$到$V_i$之间有一条边。

# Standard Output

一行一个数表示至少需要添加多少条边。
