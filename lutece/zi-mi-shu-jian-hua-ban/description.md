
# Content

相信大家都做过一道题：求水仙花数

水仙花数的定义为：如果$x$是$3$位数，$x$的各位的$3$次方的也为$x$,那么$x$为水仙花数

显然，水仙花数一共有$4$个：$153,370,371,407$

我们现在定义$n$位自幂数为：

如果$x$是$n$位数，$x$的各位的$n$次方的也为$x$,那么$x$为$n$位自幂数

求$n$位自幂数有哪些，排序后输出

请注意，这里定义的自幂数必须大于$0$

# Standard Input

一个数$n(1 \le n \le 7)$

# Standard Output

一行数，空格隔开，代表$n$位自幂数。

如果不存在，请输出$-1$
