
# Content

以后人类都不用RMB了，都是用`悦田币(YTC)`，学姐姐有$A$个面值为$1$`YTC`的硬币，和无穷个$500$`YTC`的硬币，学姐姐想要买小裙子，小裙子值$N$`YTC`，学姐姐想知道是否能准确付$N$`YTC`不找零。

# Standard Input

输入两个数表示$N,A$  
$1\le N\le 10000$  
$0\le A \le 1000$

# Standard Output

输出Yes或者No
