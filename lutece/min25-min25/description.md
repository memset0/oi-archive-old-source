
# Content

设$f(x,k) = x^k * \sigma(x^k)$,$\sigma(x)$是约数个数函数。

现在请你计算$\sum_{x=1}^{n}{f(x,k)}$在模$1e9+7$意义下的答案。

# Standard Input

一个$n$和一个$k$。

# Standard Output

输出一行代表最后的答案。
