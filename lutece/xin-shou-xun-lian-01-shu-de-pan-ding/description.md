
# Content

判断给定的无向图是不是一棵树。

# Standard Input

第一行输入$2$个数$n$, $m$，表示图有$n$个节点（编号$0$到$n-1$），$m$条边。

接下来$m$行描述$m$条边，每行$2$个数$a$, $b$，表示$a$节点和$b$节点之间有一条边相连。

$1\leq n\leq 10^5$, $0\leq m\leq 10^5$, $0\leq a,b\leq n-1$

# Standard Output

`true`，表示给定的图是一棵树，否则输出`false`。
