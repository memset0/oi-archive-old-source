
# Content

给出平面上$n$个不同的点，求有多少个平行于坐标轴的矩形

# Standard Input

第一行一个整数$n(1\le n \le10^5)$，表示点的数量  
接下来$n$行，每行有两个整数，表示点的坐标$x\_i,y\_i(-10^9\le x\_i,y\_i\le 10^9)$

# Standard Output

输出一个整数ans，表示平行于坐标轴的矩形的数量
