
# Content

求$\sum_{i=1}^{n}{i^3 + i^2 + i + 1}[i为质数]$

由于答案非常大，你只需要输出答案模$p$就行了。

# Standard Input

一个$n$,一个$p$,我们保证$p$为素数。

# Standard Output

题目要求的和在模$p$意义下的答案
