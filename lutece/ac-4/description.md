
# Content

kmp板题
给出两个不含空格的字符串 $a$ , $b$ 。
输出 $b$ 串在 $a$ 串中所有出现位置（头部对齐）。

# Standard Input

第一行一个字符串 $a$。
第二行一个字符串 $b$。

# Standard Output

一行输出所有匹配位置。
