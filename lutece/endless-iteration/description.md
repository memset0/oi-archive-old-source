
# Content

记$f(x)=2^x$。求$f^{(\infty)}(x)$在模$P$意义下的值。

# Standard Input

第一行一个整数$T$($T\leq 100000$),表示数据组数

接下来$T$行,每行一个整数$P$($T\leq 2000000000$),表示模数.

# Standard Output

共$T$行,每行一个整数,表示模$P$意义下的答案.
