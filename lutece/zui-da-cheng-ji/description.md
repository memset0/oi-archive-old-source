
# Content

输入$n$个整数元素组成的序列$S$，你需要找出一个乘积最大的连续子序列。如果这个最大的乘积不是正数，应输出-1。1<=$n$<=18, -10<=$S_i$<=10。

# Standard Input

第一行输入一个整数，代表$n$个整数

第二行是$n$个整数组成的序列$S$

# Standard Output

输出一行，代表计算出来的最大乘积。
