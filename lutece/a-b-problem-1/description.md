
# Content

Calculate $a+b$.

In this problem, $a$ and $b$ may be presented as *Octal*, *Decimal* or *Hexadecimal*.

# Standard Input

Two integer $a,b (0 \leq a,b < 16^{10000})$

Each line contains one integer.

# Standard Output

Output $a+b$ as a *Decimal* number.
