
# Content

学姐姐今天想知道$[L,R]$区间内有多少个回文数字。  
回文数字是从左向右读和从右向左读是相同的，如`123321`，`12321`，`7`。

# Standard Input

第一行输出两个数表示L，R  
$10000 \le L, R \le 99999$

# Standard Output

输出一个数表示答案个数。
