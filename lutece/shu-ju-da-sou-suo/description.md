
# Content

写一个程序，把一个字符串中的数字子序列找出来并转换成十进制整数输出。

# Standard Input

第一行是整数$n$，表示测试的数据组数，下面是$n$行无空格的字符串（长度不超过200）。题目保证字符串中含的整数位数小于$9$位，不考虑负数情形。

# Standard Output

每一行输入对应一行输出，一行中若有多个数据，每个数据后有一个空格。
