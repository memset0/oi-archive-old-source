
# Content

一群人排成一圈，按顺时针编号$1,2,3,\ldots$。然后顺时针移动$m$次，编号又是怎样的。

# Standard Input

第一行有两个整数$n$和$T$，$n$表示人数，$T$表示测试的组数。下面是$T$行整数，每行一个整数，表示移动次数$m$。$1\le n\le 20$，$0\le m\le 100$。

# Standard Output

共有$T$行，每行对应移动后的人员编号，每个编号后有一个空格。
