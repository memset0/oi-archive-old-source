
# Content

任给两个正整数，求两数的最大公约数与最小公倍数。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 1000$），表示测试数据的组数。每一组测试数据只有一行，分别为整数$a$和$b$，两数之间有一个空格。该行没有其它多余的符号。$0<a,b\le 32767$。

# Standard Output

对应每组输入，输出一行对应两数的最大公约数和最小公倍数，两数之间用一个空格隔开。该行不能有其它多余的符号。
