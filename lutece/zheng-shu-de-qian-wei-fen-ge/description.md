
# Content

读入一个整数，利用逗号进行千位分隔并输出。

# Standard Input

输入第一行是整数$T$($0<T\leq 50$)，表示接下来有$T$行，每行有一个整数，即为要利用逗号千位分隔的整数$k$，其中$0\leq k\leq 2^{31}-1$。

# Standard Output

对应每一行数据输入，输出一行用逗号进行千位分隔的数字串。
