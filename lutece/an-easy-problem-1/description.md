
# Content

Given an interger $N$, you are supposed to tell me the last digit of $S=1^1+2^2+....+N^N$.

# Standard Input

The first line contains an integer $N$ ($1 \leq N \leq 10^{100000}$).

# Standard Output

Output a digit meaning the last digit of $S=1^1+2^2+....+N^N$.
