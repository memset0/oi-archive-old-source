
# Content

给出字符串可重集$S$

任意选出两个$S$中的字符串$a$，$b$，再任意选出$a$的前缀$a'$，$b$的前缀$b'$，连接成为$c=a'+b'$

请问有多少个不同的$c$

# Standard Input

第一行一个$n(n \le 10000)$，代表$S$集合大小
接下来$n$个串，代表$S$的第$i$个串
数据保证每个串均由小写字母组成，并且每个串长度小于等于$30$

# Standard Output

一个数表示不同的$c$的个数
